# 1. 线性回归
1. 回归（regression）是能为⼀个或多个自变量与因变量之间关系建模的⼀类方法
2. 线性回归的基本元素
	- 数据集称为训练数据集（training dataset）或训练集 （training set）
	- 每行数据称为样本（sample），也可称为数据点（data point）或数据样本（data instance）
	- 预测的目标称为标签（label）或目标（target）
	- 自变量称为特征（feature）或协变量（covariate）
	- 线性模型
		- 权重（weight）
		- 偏置（bias）
		- 仿射变换（affine transformation）：通过加权和对特征进行线性变换（linear transformation），并通过偏置项来进行平移（translation）
	- 损失函数
		- 模型质量的度量方式
		- 回归问题中最常用的损失函数是平方误差函数
		- 常数1/2不会带来本质的差别，但这样在形式上稍微简单⼀些（因为当对损失函数求导后常数系数为1）
	- 解析解
		- 解可以用⼀个公式简单地表达出来：$w^* = (X^{\top}X)^{-1}X^{\top}y$
	- 随机梯度下降
		- 更新模型以提高模型预测质量的方法，计算损失函数（数据集中所有样本的损失均值）关于模型参数的导数。
		- 小批量随机梯度下降（mini batch stochastic gradient descent），计算小批量的平均损失关于模型参数的导数，加快更新速度。
		- 参数更新：将梯度乘以⼀个预先确定的正数，并从当前参数的值中减掉。
		- 批量大小和学习率：可以调整但不在训练过程中更新的参数称为超参数。
		- 调参（hyper parameter tuning）：选择超参数的过程。
	- 用模型进行预测
		- 给定特征估计目标的过程通常称为预测（prediction）或推断（inference）。
3. 向量加速，对计算进向量化，通常会带来数量级的加速
4. 正态分布与平方损失
	- 根据极大似然估计法选择的估计量称为极大似然估计量。
	- 在高斯噪声的假设下，最小化均方误差等价于对线性模型的极大似然估计。
5. 从线性回归到深度网络
	- 可将线性回归模型视为由单个人工神经元组成的神经网络，或称为单层神经网络。
	- 对于线性回归，每个输入都与每个输出相连，这种变换称为全连接层（fully connected layer）或稠密层（dense layer）。
# 2. 线性回归的从零开始实现
1. ⽣成数据集
```python
def synthetic_data(w, b, num_examples):
	"""⽣成y=Xw+b+噪声""" 
	X = torch.normal(0, 1, (num_examples, len(w))) 
	y = torch.matmul(X, w) + b 
	y += torch.normal(0, 0.01, y.shape) 
	return X, y.reshape((-1, 1))
```
2. 读取数据集
	- 定义⼀个data_iter函数，该函数能打乱数据集中的样本并以小批量方式获取数据
	- `yield`用“流”的方式逐批给出数据，每次循环调用这个函数时，只返回当前 batch，不一次性把所有数据加载到内存。内存友好，尤其适合大数据集。
```python
def data_iter(batch_size, features, labels): 
	num_examples = len(features)
	indices = list(range(num_examples)) 
	#这些样本是随机读取的，没有特定的顺序 
	random.shuffle(indices) 
	for i in range(0, num_examples, batch_size): 
		batch_indices = torch.tensor(indices[i: min(i + batch_size,                      num_examples)]) 
		yield features[batch_indices],labels[batch_indices]
```

6. 训练
	- 在每个迭代周期（epoch）中，使用data_iter函数遍历整个数据集，并将训练数据集中所有样本都使用⼀次
	- with torch.no_grad():  是 PyTorch 中的一个上下文管理器，它的作用是：在该代码块内，所有涉及张量的计算都不会被记录到计算图中，也就不会进行梯度计算。验证/推理阶段可用来减少内存占用并加速计算。

# 3. 线性回归的简洁实现
```Python
import torch
from torch import nn
from torch.utils import data
from d2l import torch as d2l
"""(1) ⽣成数据集"""
true_w =torch.tensor([2,-3.4])
true_b =4.2
features,labels= d2l.synthetic_data(true_w,true_b,1000)
"""(2) 读取数据集"""
def load_array(data_arrays, batch_size, is_train=True):
	# 构造⼀个PyTorch数据迭代器
	dataset = data.TensorDataset(*data_arrays)
	return data.DataLoader(dataset, batch_size, shuffle=is_train)
data_iter= load_array((features,labels),batch_size)
"""(3) 定义模型"""
net = nn.Sequential(nn.Linear(2, 1)) # nn.Linear(in_features=2, out_features=1, bias=True)
"""(4) 初始化模型参数"""
net[0].weight.data.normal_(0, 0.01)
net[0].bias.data.fill_(0)
"""(5) 定义损失函数"""
loss = nn.MSELoss()
"""(6) 定义优化算法"""
trainer = torch.optim.SGD(net.parameters(), lr=0.03)
"""(7) 训练"""
num_epochs = 3
for epoch in range(num_epochs):
	for X, y in data_iter:
		l = loss(net(X) ,y) # 计算损失
		trainer.zero_grad() # 梯度清零
		l.backward()        # 反向传播，计算梯度
		trainer.step()      # 参数更新
	l = loss(net(features), labels)
	print(f'epoch {epoch + 1}, loss {l:f}')
"""(8) 参数"""
w = net[0].weight.data
b = net[0].bias.data
```
# 4. softmax回归
1. 分类问题
	- 独热编码（one-hot encoding）：是⼀个向量，它的分量和类别⼀样多。类别应的分量设置为1，其他所有分量设置为0
2. 网络架构
	- 与线性回归⼀样，softmax回归也是⼀个单层神经网络，每个输出对应一个仿射函 数
	- 对于任何具有个d输入和q个输出的全连接层，参数开销为 $O(dq)$
3. 尽管softmax是⼀个非线性函数，但softmax回归的输出仍然由输入特征的仿射变换决定。因此，softmax回归是⼀个线性模型。
4. 损失函数
	- 交叉熵损失（cross-entropy loss）：$(y,\hat y) = -\sum_{j=1}^{q} y_j\log\hat y_j$
	- $y$ 为长度为 $q$ 的独热向量，故仅对应标签项保留
	- 所有 $\hat y_j$ 为预测概率，$\log\hat y_j\leq 0$；若正确预测则损失最小（趋于 $0$）
5. softmax及其导数
	- 导数 = $\frac{\partial\ell(y,\hat y)}{\partial o_j} = \hat y_j - y_j$
	- 梯度 = 模型分配概率 $\hat y_j$ 与真实标签 $y_j$（独热）之差
	- 与回归残差形式一致，便于计算
6. 熵（entropy）
	- 定义 $H[P] = -\sum_i P(i)\log P(i)$
	- 量化分布 $P$ 的信息量

# 7. softmax回归的简洁实现
```python
"""初始化模型参数"""
net = nn.Sequential(nn.Flatten(), nn.Linear(784, 10))
def init_weights(m):
	if type(m) == nn.Linear:
		nn.init.normal_(m.weight, std=0.01)
net.apply(init_weights)
"""损失函数"""
loss = nn.CrossEntropyLoss(reduction='none')
"""优化算法"""
trainer = torch.optim.SGD(net.parameters(), lr=0.1)
```
1. `nn.Flatten()` 在做什么？
	- 把输入的张量从 “图片格式” 变成 “向量格式”。
	- 例：输入形状  (B, 1, 28, 28)  → 输出  (B, 784) 。
	- 只有变成一维向量后，后面的`nn.Linear(784, 10)`才能接收
2. `reduction='none'`在做什么？
	- 告诉`CrossEntropyLoss`“不要对样本求平均/求和，把每个样本的损失单独返回”。
	- 返回形状`(B,)`的张量，每个元素对应一个样本的损失
	- 后期加权重、做掩码或看每条样本的loss，就设'none'；若只需标量做反向传播，默认'mean'即可
3. softmax数值稳定技巧
	- **上溢风险**：$\exp(o_k)$ 可能超过数据类型最大值，得到 inf或nan
	- 稳定计算：令 $m = \max_k o_k$，则$\displaystyle \hat y_j = \frac{\exp(o_j - m)}{\sum_k \exp(o_k - m)}$
	- 分子分母同乘 $\exp(-m)$，结果不变，却避免上溢
	- **下溢风险**：$\exp(o_j - m)\to 0$，因精度限制被舍入成0使 $\log\hat y_j\to-\infty$，
	- 交叉熵合并直接计算$\log\hat y_j = o_j - m - \log\sum_k\exp(o_k - m)$
	- 避开 $\exp(\cdot)$ 与 $\log(\cdot)$ 两极，数值稳定且速度更快
