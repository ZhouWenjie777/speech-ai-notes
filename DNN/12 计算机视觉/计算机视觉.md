# 0. 深度学习-计算机视觉
1. 深度学习已成为提升 CV 系统性能的核心驱动力，涵盖医疗诊断、自动驾驶、监控等关键场景
2. 聚焦 CV 前沿：先以图像增广 & 微调增强分类泛化；后利用深度分层特征展开三大任务：
	  - 目标检测（object detection）：回顾主流检测框架与组件
	  - 语义分割（semantic segmentation）：全卷积网络（FCN）实现像素级分类
	  - 风格迁移（style transfer）：借助分层表示合成风格化图像（如封面）
# 1. 图像增广
1. 翻转
	  - 左右：$50\%$ 概率 `torchvision.transforms.RandomHorizontalFlip()`，类别不变  
	  - 上下：$50\%$ 概率 `torchvision.transforms.RandomVerticalFlip()`，较少用但可行
2. 随机裁剪  
	  - `shape_aug = torchvision.transforms.RandomResizedCrop(268,268, scale=(0.1,1), ratio=(0.5,2))`：  面积均匀采样为 $10\%$–$100\%$，宽高比随机取值为 $0.5$~$2$，再缩放到 $268\!\times\!268$，降低位置敏感性。
3. 颜色扰动
	- 单通道扰动
		- 亮度：$\mathcal U(0.5,1.5)$ 倍原图；  
		- 色调：$\Delta h\sim\mathcal U(-0.5,0.5)$（相对色盘偏移）
	- 四元同时扰动（亮度、对比度、饱和度、色度）
		- `color_aug = torchvision.transforms.ColorJitter(brightness=0.5, contrast=0.5, saturation=0.5, hue=0.5)` 联合采样，扩充颜色空间
4. 组合增广
	- `augs = torchvision.transforms.Compose([RandomHorizontalFlip(), color_aug, shape_aug])` 一次前向完成翻转+颜色+裁剪，实践中默认。
# 2. 微调
1. 数据困境：目标数据集样本数量有限，直接训练复杂模型易过拟合且精度不足。
2. 迁移学习：源数据集（如ImageNet）学得的边缘、纹理等通用特征可迁移至目标域。
3. 迁移学习中常用技巧：微调（fine-tuning）：
	- 在源数据集预训练源模型。  
	- 复制除输出层外的结构与参数到目标模型。  
	- 新建输出层，其维度等于目标类别数，权重随机初始化。  
	- 在目标数据集训练：输出层从头更新，其余层微调。  
	- 适用场景：目标数据集规模远小于源数据集，可显著提升泛化能力。
4. 微调预训练
```python
"""(1) 获取数据集"""
train_imgs=torchvision.datasets.ImageFolder(os.path.join(data_dir,'train'))
test_imgs =torchvision.datasets.ImageFolder(os.path.join(data_dir, 'test'))

"""(2) 数据预处理"""
normalize = torchvision.transforms.Normalize(
	[0.485, 0.456, 0.406], [0.229, 0.224, 0.225])

train_augs = torchvision.transforms.Compose([ 
	torchvision.transforms.RandomResizedCrop(224), 
	torchvision.transforms.RandomHorizontalFlip(), 
	torchvision.transforms.ToTensor(), 
	normalize])

test_augs = torchvision.transforms.Compose([ 
	torchvision.transforms.Resize(256), 
	torchvision.transforms.CenterCrop(224), 
	torchvision.transforms.ToTensor(), 
	normalize])

"""(3) 定义和初始化模型"""
finetune_net = torchvision.models.resnet18(pretrained = True) 
finetune_net.fc = nn.Linear(finetune_net.fc.in_features, 2) 
nn.init.xavier_uniform_(finetune_net.fc.weight)

# 整个模型从头开始训练
scratch_net = torchvision.models.resnet18()
scratch_net.fc = nn.Linear(scratch_net.fc.in_features, 2)

"""(4) 微调模型"""
def train_fine_tuning(net, learning_rate, batch_size=128, num_epochs=5, 
					  param_group=True):
	# 如果param_group=True，输出层中的模型参数将使用十倍的学习率
	train_iter=torch.utils.data.DataLoader(torchvision.datasets.ImageFolder(
		os.path.join(data_dir, 'train'), transform=train_augs), 
		batch_size=batch_size, shuffle=True)
	test_iter=torch.utils.data.DataLoader(torchvision.datasets.ImageFolder(
		os.path.join(data_dir, 'test'), transform=test_augs), 
		batch_size=batch_size)
	devices = d2l.try_all_gpus() 
	loss= nn.CrossEntropyLoss(reduction="none")
	if param_group: 
		params_1x=[param for name, param in net.named_parameters() 
			if name not in ["fc.weight", "fc.bias"]]
		trainer = torch.optim.SGD([{'params': params_1x},
								   {'params': net.fc.parameters(),
									'lr': learning_rate * 10}],
								  lr = learning_rate, weight_decay = 0.001)
	else:
		trainer = torch.optim.SGD(net.parameters(), lr = learning_rate,
								 weight_decay = 0.001)
	d2l.train_ch13(net, train_iter, test_iter, loss, trainer, num_epochs,
				   devices)

"""(5) 预训练"""
train_fine_tuning(finetune_net, 5e-5) # 微调
train_fine_tuning(scratch_net, 5e-4, param_group = False) # 从头开始训练
```
5. 小结
	- 迁移学习把源数据集知识迁移到目标数据集，微调是其常用技术
	- 目标模型复制源模型全部结构与参数，仅输出层替换并随机初始化，随后在目标数据上微调
	- 微调层使用较小学习率，输出层从头训练可用更大学习率
# 3. 目标检测和边界框
1. 任务定义：同时预测图像中多个目标的类别与空间位置，又称目标识别。
2. 应用场景：无人驾驶（车辆/行人/道路）、机器人抓取、安防异常目标检测。
3. 边界框表示  
	- 两角法：$(x_{\text{left}}, y_{\text{top}}, x_{\text{right}}, y_{\text{bottom}})$  
	- 中心法：$(x_{\text{center}}, y_{\text{center}}, w, h)$  
	- 互转函数：`box_corner_to_center`, `box_center_to_corner`，输入可为长度 4 张量或形状 $(n,4)$ 二维张量。
# 4. 锚框
1. 锚框生成
	- 策略：以每个像素为中心，按预设缩放比 $s\in(0,1]$ 与宽高比 $r>0$ 生成锚框。锚框宽高：$w_\text{anchor}=ws\sqrt{r}$，$h_\text{anchor}=hs/\sqrt{r}$，中心固定则框确定。
	- 组合采样：给定 $n$ 个缩放比 $s_1,\dots,s_n$ 与 $m$ 个宽高比 $r_1,\dots,r_m$，仅取$(s_1,r_1),\dots,(s_1,r_m)$ 与 $(s_2,r_1),\dots,(s_n,r_1)$ 共 $n+m-1$ 组，避免 $whnm$ 爆炸。
	- 总量：整图锚框数 $=wh(n+m-1)$，由 `multibox_prior(img, sizes, ratios)` 一次性返回，供后续目标检测模型使用。
2. 交并比（IoU）
	- 定义：$IoU = \frac{\text{相交面积}}{\text{相并面积}} \in [0,1]$，衡量两框重叠程度；0 无重合，1 完全重合。
	- 本质：将边界框像素集合视为集合，用杰卡德（Jaccard）系数量化相似度。
	- 用途：后续用 IoU 筛选与真实框匹配的正负锚框，及 NMS 去重；`box_iou(anchors, gt_boxes)` 返回成对 IoU 矩阵。
3. 训练数据锚框标注
	- 锚框分配算法
		- 输入：锚框集合 $\{A_i\}_{i=1}^{n_a}$，真实框集合 $\{B_j\}_{j=1}^{n_b}$，$n_a\ge n_b$；构造 IoU 矩阵 $\mathbf X\in\mathbb R^{n_a\times n_b},\ X_{ij}=\text{IoU}(A_i,B_j)$。
		- 三步分配：  
			- 找全局最大 $X_{i_1j_1}$，将 $B_{j_1}$ 分配给 $A_{i_1}$，删除第 $i_1$ 行与第 $j_1$ 列。  
			- 在剩余矩阵重复“找最大-分配-删行删列”，直至所有 $n_b$ 列被处理，保证每个真实框唯一对应一个锚框。  
			- 对剩余 $n_a-n_b$ 个锚框，在对应行中找最大 IoU；若值大于预设阈值（如 0.5），则对应真实框分配给该锚框，否则标为背景
		- 输出：每个锚框获得类别标签与相对于匹配真实框的偏移量，作为训练目标
	- 锚框标签：类别与偏移量
		- 正类锚框（被分配真实框 B）
			- 类别：直接赋 B 的类别标签。
			- 偏移量：$\delta_x = 10\frac{x_b - x_a}{w_a},\quad\delta_y = 10\frac{y_b - y_a}{h_a},\quad\delta_w = 5\log\frac{w_b}{w_a},\quad\delta_h = 5\log\frac{h_b}{h_a}$，其中 $(x_a,y_a,w_a,h_a)$ 为锚框 A 的中心-宽高表示，$(x_b,y_b,w_b,h_b)$ 为真实框 B；系数10、5 为缩放系数。
		- 负类锚框（无匹配真实框）
			- 类别标为背景（索引 0），偏移量不参与损失计算
		- 实现：`multibox_target(anchors, labels)` 返回
			- 类别张量：背景为 0，正类索引依次递增
			- 偏移张量：仅正类锚框有非零值，形状与锚框数一致
4. 非极大值抑制（NMS）
	- 解码预测框
		- 对锚框 $\mathbf a$ 与预测偏移 $\hat\delta$ 执行逆变换：$x_{\text{pred}} = x_a + \hat\delta_x w_a/10,\; y_{\text{pred}} = y_a + \hat\delta_y h_a/10,\; w_{\text{pred}} = w_a \exp(\hat\delta_w/5),\; h_{\text{pred}} = h_a \exp(\hat\delta_h/5)$
		- 再由中心-宽高转两角表示，得最终预测框
	-  NMS 流程
		- 按类别无关方式，对所有非背景预测框按置信度 $p$ 降序排列得列表 $L$。  
		- 取 $L$ 中置信度最高框 $B_1$ 作为基准，移除与其 IoU > 阈值 $\epsilon$ 的所有框。  
		- 在剩余框中重复步骤 2，直至 $L$ 为空。  
		- 输出保留下来的框，保证任意两框 IoU < $\epsilon$，消除重复检测
	- 实现：`nms(boxes, scores, iou_thresh)` 返回保留框的索引序列
5. 小结
	- 以每个像素为中心，按预设缩放比与宽高比生成锚框，密集覆盖图像。
	- IoU = $\frac{\text{交集面积}}{\text{并集面积}} \in [0,1]$，用于度量锚框与真实框或预测框间的相似度。
	- 训练标签：每个锚框分配类别（含背景）及相对于匹配真实框的 4 维偏移量。
	- 预测后处理：对所有预测框按置信度排序，用 NMS 移除高 IoU 重复框，输出最终检测结果
# 5. 多尺度目标检测
1. 多尺度锚框
	- 问题：原图 561×728，每像素 5 框 → 2M+ 锚框，计算爆炸。
	- 解决：只在特征图上均匀采样中心，而非原图。特征图宽高 fmap_w×fmap_h 控制采样密度，stride 自然覆盖全图。
	- 小目标策略：  设 fmap_h=fmap_w=4，尺度 s=0.15，共 16 个中心；每中心 3 种宽高比 → 48 个锚框，密度高但总量可控。
	- 放大特征图尺寸即可提升小目标采样密度；缩小特征图则减少大目标锚框，实现“小图多框、大图少框”的多尺度均衡。
2. 多尺度检测
	- 特征图 → 锚框 → 预测：
		- 同一空间位置的 c 个通道特征通过 $1\times1$ 卷积映射为 $a×(K+4)$ 维输出，$K$ 类得分 + 4 维偏移，完成 hw×a 次预测。
	- 感受野逐级递增：
		- 浅层 stride 小、感受野小 → 负责小目标；深层 stride 大、感受野大 → 负责大目标；
		- 网络不同层级输出多组特征图，每层独立预测 hw×a 个锚框，一次前向即可输出多层多尺度检测结果。
	- 训练与损失：
		- 每层独立匹配真实框，正负锚框共同参与分类与回归损失；多尺度标签合并后反向传播，端到端优化。
# 6. 目标检测数据集
1. 香蕉数据批量格式
	- 图像批量：(B, 3, 256, 256)，与分类任务一致。
	- 标签批量：(B, m, 5)，m=1（该数据集单框）。  
	- 填充规则：若图像框数 < m，用类别 −1 的“非法框”(−1, 0, 0, 0, 0) 补齐，保证同批张量对齐。  
	- 通用检测：m 取训练集单图最大框数；不足 m 补 −1 即可实现批量高效训练。
# 7. 单发多框检测（SSD）
1. SSD 模型结构
	- 基础网络：截断 VGG（或 ResNet），输出 stride=8 的深层特征图，负责小目标，锚框密集。
	- 多尺度特征块：逐级减半高宽、加倍感受野，每层附预测头，输出较大目标所需稀疏锚框。
	- 预测头统一：每层特征图同空间位置 c 个通道 → 1×1 卷积 → 类别得分 (K+1) 与偏移 (4)，一次性完成密集检测。
2. 类别预测层
	- 输入特征图：$(h, w, c)$  
	- 1×1 卷积保持空间分辨率，输出通道数 $= α(q+1)$，其中 $q$ 为前景类别数，$α$ 为该位置每单元锚框数。
	- 通道布局：索引 $i(q+1)+j$ 对应第 $i$ 个锚框、类别 $j$ 的得分，$0\le j\le q$（0 为背景）。
	- 优势：全卷积实现，参数共享，避免全连接带来的参数量爆炸。
3. 边界框预测层 & 多尺度连结
	- 框偏移头：同位置 3×3 卷积，输出通道 = 4α，每锚框 4 维偏移；保持空间分辨率。
	- 多尺度连结 flatten 策略：  `flatten_pred` 先把通道移到最后，再展平为 `(B, h×w×4α)`；不同尺度预测可沿 dim=1 拼接。
	- 高宽减半块：  2× 3×3 conv (pad=1) + 2×2 maxpool stride=2，输出尺寸减半；堆叠后感受野 6×6，逐级扩大，适配大目标。
4. 完整 SSD 模型
	- 五模块堆叠  
		- 1：base_net，stride=8，通道 64，最小锚框
		- 2-4：down_sample_blk，高宽减半，通道 128，逐级增大锚框
		- 5 `AdaptiveMaxPool2d((1,1))`，全局池化，单像素感受野最大，负责最大目标。
	- 锚框尺度  
		- sizes 列表：\[0.20, 0.37, 0.54, 0.71, 0.88\]，每级再配较大值 $\sqrt{s_k s_{k+1}}$，共 5 组尺寸，每层负责其一
	- 前向输出
		- 每模块返回 `(特征图, 锚框, 类别预测, 偏移预测)`，经 `concat_preds` 后合并为单张量，供损失与 NMS 使用。
5. 小结
	- 单网络一次前向：多尺度特征 → 多组锚框 → 联合预测类别与偏移，无需区域提议，速度优势显著。
	- 训练损失：  
  $$
  \mathcal L = \frac{1}{N_{\text{pos}}}\Big(\sum_i \mathcal L_{\text{cls}}(p_i, \hat p_i) + \lambda \sum_i \mathbb 1_{\text{pos}} \mathcal L_{\text{smooth L1}}(\delta_i, \hat \delta_i)\Big)
  $$
	  其中正锚框数 $N_{\text{pos}}$，背景框仅参与分类，$\lambda$ 通常取 1。
	- 推断流程：网络输出 → 解码偏移得预测框 → 按类别得分降序 → 逐类 NMS → 输出最终检测框与类别。
# 8. 区域卷积神经网络（R-CNN）系列
1. R-CNN
	- 流程
		- 候选生成：选择性搜索产出 ~2000 提议区域（不同尺度/形状）
		- 特征提取：每个提议变形到固定尺寸 → 截断 CNN → 提取固定维特征向量
		- 分类：
			- 将每个提议区域的特征连同其标注的类别作为样本，训练SVM。
			- 每类一个线性 SVM，以特征向量判断是否属于该类
		- 回归：
			- 将每个提议区域的特征连同其标注的边界框作为样本，训练线性回归模型来预测真实边界框
			- 线性回归器针对每个提议微调边界框偏移
	- 瓶颈：上千次 CNN 前向，训练分阶段，磁盘缓存大，速度极慢，难以实时
2. Fast R-CNN 关键改进
	- 共享卷积：整图单次前向，得卷积特征图 `1×c×h1×w1`，避免重复计算
	- 兴趣区域汇聚层（RoI Pooling）：对任意提议区域，划分为固定 `h2×w2` 网格，每个子窗口取最大响应，输出统一 `c×h2×w2` 特征
	- 端到端：全连接后并行 softmax 分类 (`n×q`) 与 bbox 回归 (`n×4`)，一次前向完成，训练速度提升约 9×，推理 25×
	- 输入：整图 + 提议列表
	- 输出：每张图固定张量，便于 batch 训练
3. Faster R-CNN 关键改进
	- 区域提议网络（RPN）替代选择性搜索，端到端学习生成 ~300 高质量提议，速度再提 10×。  
	- RPN 流程
		- 3×3 卷积降维得通道 c 特征；
		- 每像素生成 k 锚框，用同一 c 维特征并行回归 4k 偏移 + 2k 前景/背景得分；
		- NMS 后输出 top-300 提议供 RoI Pooling。  
	- 联合损失：  
  $$
  \mathcal L = \mathcal L_\text{RPN-cls} + \lambda_1\mathcal L_\text{RPN-reg} +\mathcal L_\text{Fast-cls} + \lambda_2\mathcal L_\text{Fast-reg}
  $$
	  四项同时反向传播，RPN 与检测头共享卷积特征，实现“attention”式提议。
4. Mask R-CNN 点睛改进
	- RoI Align 替代 RoI Pooling：双线性插值保留亚像素位置，避免量化偏移，提升掩码精度。  
	- 三分支输出：
		- RPN 前景/背景 + 偏移
		- Fast-head 分类 + bbox 回归
		- mask 分支：全卷积网络输出 28×28 每像素二分类，与类别预测解耦，仅对最终检测框计算掩码损失
	- 损失函数：
		- $\mathcal L = \mathcal L_\text{Faster} + \mathcal L_\text{mask}$，其中 $\mathcal L_\text{mask}$ 为平均二值交叉熵，端到端训练，掩码标注即像素级监督。
5. 小结
	- 演进路线
		- R-CNN（独立 CNN） → Fast R-CNN（共享卷积 + RoI Pooling） → Faster R-CNN（RPN 取代搜索） → Mask R-CNN（RoI Align + 掩码分支）
	- 核心趋势
		- 逐步把外部模块融入网络，减少候选数量，提升精度与速度，最终端到端训练
	- 应用扩展
		- Faster 后成为两阶段检测标准基线；Mask版本支撑实例分割、人体关键点、目标追踪等像素级任务
# 9. 语义分割和数据集
1. 分割三任务辨析
	- 图像分割：仅按像素相似性分组，无语义标签，可能把黑狗嘴与黄狗身切开成两区
	- 语义分割：逐像素分类到语义类别，如“狗”“猫”“背景”，但不区分同类不同实例（两条狗同标签）
	- 实例分割：在语义基础上再分实例，输出每条狗独立的像素掩码，等价于“检测+分割”
2. 小结
	- 目标：为每个像素分配语义类别，输出与输入同分辨率的分割图。
	- 数据集：Pascal VOC2012 提供 20 类目标（如人、车、狗、椅子等）和 1 类背景（非目标区域）像素级标注，是语义分割标准基准。
	- 预处理：训练阶段采用随机裁剪到固定尺寸（如 320×320）而非缩放，保持像素-标签一一对应；推断时用滑动窗口或全卷积一次前向。
# 10. 转置卷积
1. 转置卷积（transposed convolution）
	- 作用：上采样恢复空间分辨率，使输出与输入同高宽，便于像素级预测。
	- 等价视角：标准卷积的梯度反向传播算子，故又名“反卷积”（非数学可逆）
	- 输出尺寸：
		- 给定 stride $s$、padding $p$，输入高 $h$，核高 $k$，输出高  
  $$
  h' = (h-1)s + k - 2p
  $$
		- 输出宽同理，可精确上采样到目标分辨率。
	- 无 pad、stride=1 简化情形：输入 $n_h×n_w$，卷积核 $k_h×k_w$；每个输入元素乘以核得到 $(n_h+k_h-1)×(n_w+k_w-1)$ 中间张量，按位置累加得最终输出。
```python
def trans_conv(X, K):
	h, w = K.shape
	Y = torch.zeros((X.shape[0] + h - 1, X.shape[1] + w - 1))
	for i in range(X.shape[0]):
		for j in range(X.shape[1]):
			Y[i:i+h, j:j+w] += X[i, j] * K
	return Y
```
2. 转置卷积细节
	- 填充作用在输出：指定 `padding=p` 会裁剪输出四周各 p 像素，等价于“反填充”。  
	- 步幅作用在中间：stride 越大，中间张量被撒得越稀疏，最终输出尺寸按比例放大。  
	- 多通道：与常规卷积一致；输入 $c_{in}$ 通道，每组 $c_{in}×k×k$ 核，输出 $c_{out}$ 通道需 $c_{out}$ 组核，可完成通道升/降。  
	- 可逆验证：若 `Y = conv(X)`，则 `convT(Y)` 与 `X` 同形（非数值相等），保证上采样对齐。
3. 转置卷积与矩阵视角
	- 卷积正向：稀疏矩阵 $\mathbf W$ 将输入向量 $\mathbf x$ 映射为 $\mathbf y=\mathbf W\mathbf x$，$\mathbf W$ 形状 $(m,n)$。  
	- 反向传播：梯度 $\nabla_{\mathbf x}=\mathbf W^\top\nabla_{\mathbf y}$，恰好是 $\mathbf W^\top$ 的矩阵乘法。  
	- 转置卷积即“交换正反向”：前向执行 $\mathbf W^\top\mathbf z$，等价于上采样；因此得名 transposed convolution，而非可逆运算。  
	- 实现：框架把标准卷积的反向传播算子暴露为前向操作，完成上采样。
4. 小结
	- 作用：通过卷积核“广播”输入元素，实现空间维度放大，完成上采样。
	- 形状复原：若 `Y = f(X)`，则同超参数转置卷积层 `g` 满足 `g(Y)` 与 `X` 形状一致，便于构建编码-解码对称结构。
	- 矩阵视角：实质是卷积正向矩阵 $\mathbf W$ 的转置 $\mathbf W^\top$ 的乘法操作，交换了前向与反向传播算子。
5. `nn.ConvTranspose2d`参数速查表

| 参数             | 类型 / 默认               | 含义                  | 常用小技巧                                                     |
| -------------- | --------------------- | ------------------- | --------------------------------------------------------- |
| in_channels    | int                   | 输入特征图通道数            | 与编码阶段对应通道保持一致                                             |
| out_channels   | int                   | 输出特征图通道数            | 决定下一层输入；解码末层通常 = 类别数                                      |
| kernel_size    | int 或 (kH, kW)        | 卷积核空间大小             | 奇数（3,5,7）便于 same-padding 计算                               |
| stride         | int 或 (sH, sW) 默认 1   | 上采样步长               | 设 2 即“2× 上采样”；可堆叠实现 4×、8×                                 |
| padding        | int 或 (pH, pW) 默认 0   | 输出四周裁剪量             | 设  p = k//2  可让输入输出高宽相同（s=1）                              |
| output_padding | int 或 (opH, opW) 默认 0 | 解决 stride>1 时输出尺寸歧义 | 一般设为  (stride-1)  以内；PyTorch 自动检查                         |
| dilation       | int 或 (dH, dW) 默认 1   | 空洞转置卷积              | 增大感受野，但参数不变；较少使用                                          |
| groups         | int 默认 1              | 分组转置卷积              | in_channels 须整除 groups；groups=in_channels → DepthWise 上采样 |
| bias           | bool 默认 True          | 是否加可学习偏置            | 后面接 BN 时可设 False                                          |
| padding_mode   | 'zeros' 默认            | 仅对 padding>0 时有效    | 通常保持 'zeros'                                              |
注：s=2 即 2× 上采样；padding 在这里是裁剪；output_padding 只在 stride>1 时调
# 11. 全卷积网络
1. 全卷积网络（FCN）模型结构
	- 结构三步走：
		- 下采样：用预训练 CNN（如 ResNet-18 去掉全局池化&FC）得 1/32 特征图
		- 通道映射：1×1 卷积把 512 → 21 通道（Pascal VOC 20+1 背景）
		- 上采样：转置卷积 stride=32、kernel=64、padding=16，一次性把 10×15 → 320×480，恢复像素级预测。
	- **上采样放大 s 倍参数设置**：`kernel_size = 2s，stride = s，padding = s // 2`，即可一次性将特征图高宽放大 $s$ 倍
	- 初始化技巧：
		- 转置卷积核可用双线性插值权重初始化
		- 对于1×1 卷积层，使用Xavier初始化
	- 特点：全卷积无全连接，任意尺寸输入输出；跳跃连接可融合中层细节提升边缘精度
2. 双线性插值初始化转置卷积
	- 用途：将转置卷积核初始为双线性权重，使上采样结果与经典双线性插值完全一致，训练初期即可得到平滑放大效果。
	- 核心公式：  对于 kernel_size=k，中心偏移 center=(k−1)/2，二维权重
	$$
  w(i,j)=\left(1-\frac{|i-\text{center}|}{f}\right)\left(1-\frac{|j-\text{center}|}{f}\right),\quad f=\left\lfloor\frac{k+1}{2}\right\rfloor
  $$
	- 使用方式：  `conv_t.weight.data = bilinear_kernel(in_c, out_c, k)`，随后可选冻结或继续微调
	- 放大 2× 示例：`k=4, stride=2, pad=1`，输出尺寸恰好 2× 输入，且初始放大无 checkerboard
	- Xavier 随机初始化对比：双线性初始化→“开局即平滑”，Xavier 随机→“开局即噪声”，后者需更多训练轮次和数据，但上限可能更高
# 12. 风格迁移
1. 风格迁移小结
	- 三张图：内容图、风格图、合成图（可初始化为内容图，唯一可训练变量）。
	- 预训练CNN固定参数，充当特征提取器；选取中间层：
		- 内容层（如`conv4_2`）：保留物体形状；
		- 风格层（如`conv1_1, conv2_1, conv3_1, conv4_1`）：捕获纹理与色彩。
	- 损失三件套：
		- 内容损失：合成图与内容图在内容层特征图的L2距离；
		- 风格损失：合成图与风格图在风格层格拉姆（Gram）矩阵的L2距离；
		- 总变差损失：合成图相邻像素差值L1，抑制噪点。
	- 优化：对合成图像像素（而非网络权重）做梯度下降，迭代更新直至损失收敛。
2. 特征抽取实现细节
	- 预训练 VGG-19 截断：只保留到最深层风格层（索引 28）之前，避免多余计算。  
	- 内容层：第四卷积块末尾 `conv4_2`（索引 25），保留物体整体形状，抑制细节。  
	- 风格层：5 个卷积块的第一个卷积层 `[0, 5, 10, 19, 28]`，兼顾局部纹理与全局色彩。  
	- 预缓存：内容与风格图像的特征只需计算一次，训练阶段仅对合成图像重复前向。
3. 损失函数汇总
	- 内容损失：仅对内容层输出计算 L2，保证物体形状不变
	- 风格损失：
		- 对每层风格特征分别计算 Gram 矩阵 L2 后求和，捕获纹理与色彩分布
		- 格拉姆矩阵：给定特征张量 $X ∈ ℝ^{C×H×W}$， reshape 为 $F ∈ ℝ^{C×HW}$，则
 $$
  G = \frac{1}{HW} F F^\top \quad ∈ ℝ^{C×C}
  $$
		  元素 $G_{ij}$ 表示通道 i 与通道 j 的特征相关性，刻画图像风格。
	- 全变分损失（TV Loss）：
		- 作用：惩罚高频跳变，抑制亮/暗颗粒噪点，使输出更平滑
		- 定义：对合成图像 $Y_{hat}$，逐像素计算相邻差分
 $$
  \mathcal L_\text{tv}= \sum_{i,j}\left(|c_{i,j}-c_{i+1,j}| + |c_{i,j}-c_{i,j+1}|\right)
  $$
		  其中 $c_{i,j}$ 为坐标 `(i,j)` 的像素值
	- 总损失：
 $$
  \mathcal L = \alpha\mathcal L_\text{content} + \beta\sum_i\mathcal L_\text{style}^i + \gamma\mathcal L_\text{tv}
  $$
	  典型权重：$\alpha=1,\ \beta=1\times10^3,\ \gamma=10$，可按视觉效果微调。
