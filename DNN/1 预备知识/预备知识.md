# 1. 概述
1. 机器学习中的关键组件
	- 数据：样本（特征-标签对）
	- 模型：数据转换
	- 目标函数：损失函数，优化目标
	- 算法：梯度下降（gradient descent），调整模型参数以优化目标函数
2. 数据集通常可以分成两部分：训练数据集用于拟合模型参数，测试数据集用于评估拟合的模型
3. 过拟合：当一个模型在训练集上表现良好，但不能推广到测试集时，这个模型被称为过拟合（Overfitting）
4. 各种机器学习问题
	- 监督学习：样本（特征-标签对）
		- 回归（regression）：当标签取任意数值时
		- 分类（classification）：当标签取有限值时
		- 标记问题
		- 搜索：排序
		- 推荐系统
		- 序列学习：标记和解析、自动语音识别、文本到语音
	- 无监督学习
		- 聚类（clustering）：没有标签的情况下数据分类
		- 主成分分析（principal component analysis）：找到少量的参数来准确地捕捉数据的线性相关属性
		- 因果关系（causality）和概率图模型（probabilistic graphical models）
		- 生成对抗性网络（generative adversarial networks）：合成数据的方法
	- 强化学习
		- 智能体（agent）
		- 环境（environment）
		- 观察（observation）
		- 动作（action）
		- 奖励（reward）
		- 强化学习框架的通用性十分强大。例如，我们可以将任何监督学习问题转化为强化学习问题。
		- 当环境可被完全观察到时，强化学习问题被称为马尔可夫决策过程（markov decision process）
# 2. 数据操作
1. 入门
	- 创建⼀个行向量：`x = torch.arange(12)`
	- 访问张量的形状：`x.shape`
	- 张量中元素总数：`torch.Size(x)`
	- 改变⼀个张量形状：`X = x.reshape(3, 4)`
	- 所有元素都设置为0：`torch.zeros((2,3,4))`
	- 所有元素都设置为1：`torch.ones((2,3,4))`
	- 标准正态分布随机采样：`torch.randn(3, 4)`
2. 运算符
	- 对于任意具有相同形状的张量，常见的标准算术运算符（+、-、\*、/和\*\*）都可以被升级为按元素运算
	- 多个张量连结（concatenate）在⼀起：`torch.cat((X,Y),dim=0)`
	- 通过逻辑运算符构建张量：`X == Y`
	- 对张量中的所有元素求和：`X.sum()`
3. 广播机制
	- 形状不同， 通过广播机制（broadcasting mechanism）来按元素操作，通过适当复制元素来扩展⼀个或两个数组，以便在转换之后，两个张量具有相同的形状，在大多数情况下，我们将沿着数组中长度为1的轴进行广播。
4. 索引和切片
	- 第⼀个元素索引是0，最后⼀个元素索引是-1；可以指定范围以包含第⼀个元素和最后⼀个之前的元素。`X[-1], X[1:3]`
	- 为多个元素赋相同的值，索引所有元素然后赋值。`X[0:2, :] = 12`
5. 节省内存
	- 执行原地操作：`X[:] = X + Y或X += Y`
6. 转换为其他Python对象
	- `a = X.numpy()`或`b = torch.tensor(a)`
	- 将大小为1的张量转换为标量，可以调用item函数或Python的内置函数。`b.item(), float(b), int(b)`

# 4. 线性代数
1. 标量
	- 仅包含⼀个数值的叫标量（scalar）
2. 向量
	- 标量值组成的列表。这些标量值称为向量的元素（element）或分量（component）
	- 向量或轴的维度表示向量或轴的长度，即向量或轴的元素数量。
	- 张量的维度表示张量具有的轴数。在这个意义上，张量的某个轴的维数就是这个轴的长度
3. 矩阵
	- 具有两个轴的张量，当矩阵具有相同数量的行和列时，称为方阵（square matrix）
	- 对称矩阵（symmetric matrix）A等于其转置：$A = A^⊤$
4. 张量
	- 具有任意数量轴的n维数组。向量是⼀阶张量，矩阵是二阶张量
	- 两个矩阵的按元素乘法称为哈达玛积（Hadamard product）（数学符号⊙）
	- 张量乘以或加上⼀个标量不会改变张量的形状，其中张量的每个元素都将与标量相加或相乘
5. 降维
	- 求和函数会沿所有的轴降低维度，使它变为标量。`A.sum()`
	- 指定轴降低维度。`A.sum(axis=0), A.sum(axis=[0, 1])`
	- 调用函数来计算任意形状张量的平均值。`A.mean()`
	- 指定轴降低维度。`A.mean(axis=0)`
	- 非降维求和。`sum_A = A.sum(axis=1, keepdims=True)`
6. 点积（Dot Product）
	- $x^⊤y或⟨x，y⟩$，两个向量相同位置元素乘积的和。`torch.dot(x, y)`
7. 矩阵-向量积
	- 可以把⼀个矩阵m x n和向量$R^n$积看作是从$R^n$到$R^m$向量的转换。这些转换是非常有用。例如，我们可以用方阵的乘法来表示旋转。`torch.mv(A,x)`
8. 矩阵-矩阵乘法
	- `torch.mm(A,B)`
9. 范数（norm）
	- ⼀个向量的范数告诉我们⼀个向量有多大。 这里的大小是分量的大小。
	- 在线性代数中，向量范数是将向量映射到标量的函数$f$。
		- 齐次性：按常数因子缩放所有元素，范数也会按相同常数因子的绝对值缩放。$f(\alpha\mathbf{x}) = |\alpha|\,f(\mathbf{x})$
		- 三角不等式：$f(\mathbf{x} + \mathbf{y}) \le f(\mathbf{x}) + f(\mathbf{y})$
		- 非负性：$f(\mathbf{x}) \ge 0$
		- 确定性（零向量当且仅当零范数）：$\forall i, [\mathbf{x}]_i = 0 \iff f(\mathbf{x}) = 0$
	- $L_2$范数：$∥x∥或∥x∥_2$，欧几里得距离，元素平方和的平方根。`torch.norm(u)`
	- $L_1$范数：向量元素的绝对值之和。`torch.abs(u).sum()`
	- $L_p$范数：$\|\mathbf{x}\|_p = \left( \sum_{i=1}^{n} |x_i|^p \right)^{\!1/p}$
# 5. 微积分
1. 导数和微分
	- 如果的导数存在，这个极限被定义为：$f'(\alpha) = \lim_{h \to 0} \frac{f(\alpha + h) - f(\alpha)}{h}$，如果f'(a)存在，则称在a处是可微（differentiable）的。
	- 导数的等价符号：$f'(\alpha)=y'=\frac{\mathrm{d}y}{\mathrm{d}x} =\frac{\mathrm{d}f}{\mathrm{d}x}=\frac{\mathrm{d}}{\mathrm{d}x}f(\alpha)=Df(\alpha)=D_{x}f(\alpha)$
	- 微分法则：常数相乘法则、加法法则、乘法法则、除法法则
2. 偏导数：将微分的思想推广到多元函数，$\frac{\partial y}{\partial x_i}= \frac{\partial f}{\partial x_i}= \partial_i f= f_i= D_i f= D_{x_i} f$
3. 梯度：⼀个多元函数对其所有变量的偏导数，得到该函数的梯度（gradient）向量。
4. 假设x为n维向量，多元函数微分规则
	- $\forall A\in\mathbb{R}^{m\times n},\quad \nabla_{\!\!x}(A x) = A^{\top}$
	- $\forall A\in\mathbb{R}^{n\times m},\quad \nabla_{\!\!x}(x^{\top} A) = A$
	- $\forall A\in\mathbb{R}^{n\times n},\quad \nabla_{\!\!x}(x^{\top} A x) = (A + A^{\top}) x$
	- $\nabla_{\!\!x}\|x\|^{2} = \nabla_{\!\!x}(x^{\top} x) = 2 x$
5. 链式法则：假设y=f(u)和u=g(x)可微，$\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}$
# 6. 自动微分
1. ⼀个标量函数关于向量x的梯度是向量，并且与x具有相同的形状。
```python
x.requires_grad_(True) # 等价于x=torch.arange(4.0,requires_grad=True)
y.backward()
x.grad
# 在默认情况下，PyTorch会累积梯度，计算x的另⼀个函数前需要清除之前的值
x.grad.zero_()
```
2. 非标量变量的反向传播。当y不是标量时，向量y关于向量x的导数的最自然解释是⼀个矩阵。
```python
# 对非标量调用backward需要传入⼀个gradient参数，该参数指定微分函数关于self的梯度。 # 在我们的例字中，我们只想求偏导数的和，所以传递⼀个1的梯度是合适的
x.grad.zero_()
y = x * x
y.sum().backward() # 等价于y.backward(torch.ones(len(x)))
x.grad
```
3. 分离计算。将某些计算移动到记录的计算图之外，如分离y来返回⼀个新变量u，梯度不会向后流经u，将u作为常数处理
```python
x.grad.zero_()
y = x * x
u = y.detach()
z = u * x
z.sum().backward()
x.grad == u # True
```
# 7. 概率
1. 基本概率论
	- 大数定律（law of large numbers）：随着投掷次数的增加，这个估计值会越来越接近真实的潜在概率
	- 从概率分布中抽取样本的过程称为抽样（sampling）
	- 将概率分配给⼀些离散选择的分布称为多项分布 （multinomial distribution）
```python
from torch.distributions import multinomial
fair_probs = torch.ones([6]) / 6 
multinomial.Multinomial(1, fair_probs).sample() # 抽一个样本
```
2. 概率论公理
	- 公理1：非负性，$P(A) \geq 0$
	- 公理2：归一性，$P(S) = 1$
	- 公理3：可数可加性，对任意互斥事件序列，$P\!\left(\bigcup_{i=1}^{\infty} A_i\right) = \sum_{i=1}^{\infty} P(A_i)$
3. 随机变量：是把样本空间映射到实数的函数
4. 多个随机变量
	- 联合概率：$P(A = a, B = b)$
	- 条件概率：$P(B = b \mid A = a)$
	- 贝叶斯定理：
		- 乘法法则 $P(A,B) = P(B|A)P(A) = P(A|B)P(B)$
		- Bayes公式 $P(A|B) = \frac{P(B|A)P(A)}{P(B)}$，其中 $P(B)>0$
	- 边际化：$P(B) = \sum_{A} P(A,B)$，结果称为边际概率或边际分布
	- 独立性：
		- 定义：$P(A|B) = P(A) \iff P(A,B) = P(A)P(B)$
		- A与B独立当且仅当联合分布等于边缘分布的乘积
		- 条件独立：给定C，若$P(A,B|C) = P(A|C)P(B|C)$，则A与B条件独立
	- 期望和方差
		- 离散随机变量期望 $E[X] = \sum_{a} a\,P(X = a)$
		- 函数期望 $E_{a\sim P}[f(a)] = \sum_{a} f(a)\,P(a)$
		- 方差 $\mathrm{Var}[X] = E\bigl[(X - E[X])^2\bigr] = E[X^2] - (E[X])^2$
		- 标准差 $\sigma_X = \sqrt{\mathrm{Var}[X]}$
		- 函数方差 $\mathrm{Var}[f(a)] = E\bigl[(f(a) - E[f(a)])^2\bigr]$
