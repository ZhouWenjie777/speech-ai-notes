# 1. 从全连接层到卷积
1. 卷积神经网络（convolutional neural network，CNN）是⼀类强大的、为处理图像数据而设计的神经网络。参数少于全连接，易用GPU并行计算。
2. 平移不变性（translation invariance）
	- 神经网络的前几层应该对相同的图像区域具有相似的反应。
	- 输入$X$中的平移，仅导致隐藏表示H中的平移，即核$V$与位置无关，偏置$U$为常数$u$
	- $[H]_{i,j} = u + \sum_a\sum_b [V]_{a,b}\cdot [X]_{i+a,j+b}$
3. 局部性（locality）
	- 神经网络的前几层应该只探索局部区域，而不过度关注相隔较远区域的关系。
	- 局部性约束：当 $|a|>\Delta$ 或 $|b|>\Delta$ 时 $[V]_{a,b}=0$
	- 卷积输出 $[H]_{i,j}=u+\sum_{a=-\Delta}^{\Delta}\sum_{b=-\Delta}^{\Delta}[V]_{a,b}\,[X]_{i+a,j+b}$
	- 卷积核 $V$ 为可学习滤波器，参数共享且平移不变
	- 参数从亿级降至百级，归纳偏置：平移不变+局部感知；若数据不满足该偏置，模型可能欠拟合
4. 卷积数学定义
	- 连续形式：$(f*g)(x)=\int_{\mathbb{R}^d}f(z)\,g(x-z)\,dz$
	- 离散形式：$(f*g)(i)=\sum_{a\in\mathbb{Z}}f(a)\,g(i-a)$
	- 核心操作：翻转 g 并滑动，测量与 f 的重叠
5. 多通道三维卷积
	- 输入与隐藏表示均为三维张量：空间高×宽×通道
	- 每个空间位置输出一组特征映射（feature maps）（通道），例如底层通道可专门识别边缘或纹理
	- 卷积核增加第4维：$[V]_{a,b,c,d}$，其中$d$为输出通道索引
	- 输出定义 $[H]_{i,j,d}=u_d+\sum_{c}\sum_{a=-\Delta}^{\Delta}\sum_{b=-\Delta}^{\Delta}[V]_{a,b,c,d}\,[X]_{i+a,j+b,c}$
	- 后续层继续以三维张量$H$作为输入，实现深层多通道特征提取
# 2. 图像卷积
1. 严格来说，卷积层是个错误的叫法，因为它所表达的运算其实是互相关运算（cross-correlation）。由于卷积核是从数据中学习到的，因此无论执行严格的卷积运算还是互相关运算，卷积层的输出都不会受到影响。
2. 互相关输出尺寸
	- 输出大小 = 输入高宽 - 卷积核高宽 + 1
	- 公式：$(n_h - k_h + 1) \times (n_w - k_w + 1)$
	- 结果：有效区域逐点互相关，无填充时特征图缩小
```python
def corr2d(X,K):
"""计算三维互相关运算"""
	h, w = K.shape
	Y = torch.zeros((X.shape[0]- h + 1, X.shape[1]- w + 1))
	for i in range(Y.shape[0]):
		for j in range(Y.shape[1]):
			Y[i, j] = (X[i:i + h, j:j + w] * K).sum()
	return Y
```
3. 卷积层：对输入和卷积核权重进行互相关运算，并在添加标量偏置之后产生输出
	- 有时被称为特征映射（feature map），被视为输入映射到下一层的空间维度的转换器
	- 某⼀层的任意元素，其感受野（receptive field）是指在前向传播期间可能影响其计算的所有元素（来自所有先前层）
```python
class Conv2D(nn.Module):
	def __init__(self, kernel_size):
		super().__init__()
		self.weight = nn.Parameter(torch.rand(kernel_size))
		self.bias = nn.Parameter(torch.zeros(1))
	
	def forward(self, x):
		return corr2d(x, self.weight) + self.bias
```

# 3. 填充和步幅
1. 填充（padding）：增加输出的高和宽
	- 输出形状 $(n_h - k_h + p_h + 1) \times (n_w - k_w + p_w + 1)$
	- 若设 $p_h = k_h - 1,\ p_w = k_w - 1$，则输入输出高宽相同
	- 奇数核方便两侧对称填充，常用 $k = 3,5,7$ 配 $p = 1,2,3$
	- 满足奇数核+对称填充+输入输出同尺寸时，$Y[i,j]$ 以 $X[i,j]$ 为中心做互相关
2. 步幅（stride）：减小输出的高和宽
	- 通用公式 $\left\lfloor\dfrac{n_h - k_h + p_h + s_h}{s_h}\right\rfloor \times \left\lfloor\dfrac{n_w - k_w + p_w + s_w}{s_w}\right\rfloor$
	- 若$p_h = k_h - 1,\ p_w = k_w - 1$，则简化为 $\left\lfloor\dfrac{n_h + s_h - 1}{s_h}\right\rfloor \times \left\lfloor\dfrac{n_w + s_w - 1}{s_w}\right\rfloor$
	- 继若 $n_h,\,n_w$ 可被 $s_h,\,s_w$ 整除时，输出形状即为 $(n_h / s_h) \times (n_w / s_w)$
3. 对于音频信号，步幅2说明“每隔一个样本取一次”，相当于把原始音频下采样到原来的一半长度（采样率 / 2）
	- 时间分辨率减半：1 s 的 16 kHz 信号变成 8 kHz
	- 每帧跳 2 个采样点，卷积核只覆盖非重叠片段
	- 计算量减半，感受野（时间跨度）翻倍，但高频信息会混叠；必须配合抗混叠滤波器（低通）或改用带 stride 的转置卷积 / 子带编码
# 4. 输入输出多通道
1. 输入多通道：卷积核形状 $c_i \times k_h \times k_w$
```python
def corr2d_multi_in(X, K): 
	# 先遍历“X”和“K”的第0个维度（通道维度），再把它们加在⼀起 
	return sum(corr2d(x, k) for x, k in zip(X, K))
```
2. 输出多通道：卷积核形状 $c_o \times c_i \times k_h \times k_w$
```python
def corr2d_multi_in_out(X, K):
	# 迭代“K”的第0个维度，每次都对输⼊“X”执⾏互相关运算。
	# 最后将所有结果都叠加在⼀起
	return torch.stack([corr2d_multi_in(X, k) for k in K], 0)
```
3. 1x1卷积层
	- 通常用于调整网络层的通道数量和控制模型复杂性
	- 可以看作是在每个像素位置应用全连接层，以$c_i$个输入值转换为$c_o$个输出值
```python
def corr2d_multi_in_out_1x1(X, K):
"""使用全连接层实现卷积"""
	c_i, h, w = X.shape
	c_o = K.shape[0]
	X = X.reshape((c_i, h * w))
	K = K.reshape((c_o, c_i))
	# 全连接层中的矩阵乘法
	Y = torch.matmul(K, X)
	return Y.reshape((c_o, h, w))
```
# 5. 汇聚层（池化层）
1. 目的/优点：降低卷积层对位置的敏感性，同时降低对空间降采样表示的敏感性。
2. 汇聚层不包含参数。通常计算汇聚窗口中所有元素的最大值或平均值。这些操作分别称为最大汇聚层（maximum pooling）和平均汇聚层（average pooling）
```python
def pool2d(X, pool_size, mode='max'):
	p_h, p_w = pool_size
	Y = torch.zeros((X.shape[0]- p_h + 1, X.shape[1]- p_w + 1))
	for i in range(Y.shape[0]):
		for j in range(Y.shape[1]):
			if mode == 'max':
				Y[i, j] = X[i: i + p_h, j: j + p_w].max()
			elif mode == 'avg':
				Y[i, j] = X[i: i + p_h, j: j + p_w].mean()
	return Y
```
3. 填充和步幅
	- 同[[卷积神经网络#3. 填充和步幅|填充和步幅]]，默认情况下，深度学习框架中的步幅与汇聚窗口的大小相同
4. 多通道
	- 在处理多通道数据时，汇聚层在每个通道上单独运算。这意味着汇聚层的输出通道数与输入通道数相同。

# 6. 卷积神经网络（LeNet）
1. 卷积神经网络
	- LeNet是最早发布的卷积神经网络之一，包括卷积编码器与全连接层密集块两个部分。
	- 设计模式：逐级降低空间分辨率，同时增加通道数
	- 传统结构：卷积块提取特征 → 一个或多个全连接层输出最终表征
```python
from torch import nn
net = nn.Sequential(
	nn.Conv2d(1, 6, kernel_size=5, padding=2), nn.Sigmoid(), 
	nn.AvgPool2d(kernel_size=2, stride=2), 
	nn.Conv2d(6, 16, kernel_size=5), nn.Sigmoid(), 
	nn.AvgPool2d(kernel_size=2, stride=2), 
	nn.Flatten(), 
	nn.Linear(16 * 5 * 5, 120), nn.Sigmoid(), 
	nn.Linear(120, 84), nn.Sigmoid(), 
	nn.Linear(84, 10))

# 在每一层打印输出的形状
X = torch.rand(size=(1, 1, 28, 28), dtype=torch.float32)
for layer in net:
	X = layer(X)
	print(layer.__class__.__name__,'output shape: \t',X.shape)
```

def train_ch6(net, train_iter, test_iter, num_epochs, lr, device):
"""⽤GPU训练模型"""
	def init_weights(m):
		if type(m) == nn.Linear or type(m) == nn.Conv2d:
			nn.init.xavier_uniform_(m.weight) # m.bias默认初始化为0
	net.apply(init_weights)
	net.to(device)
	optimizer = torch.optim.SGD(net.parameters(), lr=lr)
	loss = nn.CrossEntropyLoss()
	animator = d2l.Animator(xlabel='epoch', xlim=[1, num_epochs], 
							legend=['train loss', 'train acc', 'test acc'])
	timer, num_batches = d2l.Timer(), len(train_iter)
	for epoch in range(num_epochs):
		metric = d2l.Accumulator(3) # 训练损失之和，训练准确率之和，样本数
		net.train()
		for i, (X, y) in enumerate(train_iter):
			timer.start()
			optimizer.zero_grad()
			X, y = X.to(device), y.to(device)
			y_hat = net(X)
			l = loss(y_hat, y)
			l.backward()
			optimizer.step()
			with torch.no_grad():
				metric.add(l*X.shape[0], d2l.accuracy(y_hat, y), X.shape[0])
			timer.stop()
			train_l = metric[0] / metric[2]
			train_acc = metric[1] / metric[2]
			if (i + 1) % (num_batches // 5) == 0 or i == num_batches- 1:
				animator.add(epoch + (i + 1) / num_batches, 
							(train_l, train_acc, None))
		test_acc = evaluate_accuracy_gpu(net, test_iter)
		animator.add(epoch + 1, (None, None, test_acc))

- `Accumulator`
```python
class Accumulator:
"""在n个变量上累加"""
	def __init__(self,n):
		self.data=[0.0] * n 
	
	def add(self, *args):
		self.data=[a + float(b) for a,b in zip(self.data, args)]
	
	def reset(self):
		self.data=[0.0] * len(self.data) 
	
	# 只要给自定义类加上__getitem__，就能把实例变成「可下标对象」
	def __getitem__(self, idx): 
		return self.data[idx]
```

def accuracy(y_hat,y): 
"""计算预测正确的数量""" 
	if len(y_hat.shape) > 1 and y_hat.shape[1] > 1: 
		y_hat= y_hat.argmax(axis=1) 
	cmp = y_hat.type(y.dtype) == y
	return float(cmp.type(y.dtype).sum())
