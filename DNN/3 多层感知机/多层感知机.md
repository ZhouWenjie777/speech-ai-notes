# 1. 多层感知机
1. 将许多全连接层堆叠在⼀起。每⼀层都输出到上面的层，直到生成最后的输出。我们可以把前L-1层看作表示，把最后⼀层看作线性预测器。这种架构通常称为多层感知机（multilayer perceptron），缩写为MLP。
2. 输入层不涉及任何计算，只需要实现隐藏层和输出层的计算。
3. 仿射函数的仿射函数还是仿射函数，因此在仿射变换之后对每个隐藏单元应用非线性的激活函数（activation function），防止多层感知机退化成线性模型。
4. 激活函数
	- ReLU函数：$\mathrm{ReLU}(x) = \max(x,\,0)$
	- sigmoid函数
		- 定义：$\mathrm{sigmoid}(x) = \dfrac{1}{1+\exp(-x)}$
		- 值域 $(0,1)$，将任意实数压缩到概率尺度
		- 当输入为0时，sigmoid函数的导数达到最大值0.25
	- tanh(双曲正切)
		- 定义：$\tanh(x) = \dfrac{1-\exp(-2x)}{1+\exp(-2x)} = \dfrac{\exp(x)-\exp(-x)}{\exp(x)+\exp(-x)}$
		- 值域 $(-1,1)$，零中心化输出
		- 当输入为0时，tanh函数的导数达到最大值1
# 2. 多层感知机的从零开始实现
```python
"""初始化模型参数"""
W1 = nn.Parameter(torch.randn(num_inputs, num_hiddens, requires_grad=True)      * 0.01) # 把N(0,1)的随机数变小成N(0,0.01²) 让初始权重接近0，避免饱和或梯度爆炸
b1 = nn.Parameter(torch.zeros(num_hiddens, requires_grad=True))
params = [W1, b1, W2, b2]
"""激活函数"""
def relu(X):
	a = torch.zeros_like(X)
	return torch.max(X, a)
"""模型"""
def net(X):
	X = X.reshape((-1, num_inputs))
	H = relu(X@W1 + b1) # 这⾥“@”代表矩阵乘法
	return (H@W2 + b2)
```

# 4. 模型选择、欠拟合和过拟合
1. 将模型在训练数据上拟合的比在潜在分布中更接近的现象称为过拟合（overfitting），用于对抗过拟合的技术称为正则化（regularization）
2. 训练误差（training error）是指，模型在训练数据集上计算得到的误差。
3. 泛化误差（generalization error）是指，模型应用在同样从原始样本的分布中抽取的无限多数据样本时，模型误差的期望。在实际中，用⼀个独立的测试集来估计泛化误差
4. 通常对于神经网络，我们认为需要更多训练迭代的模型比较复杂。
5. 影响模型泛化的因素
	- 可调整参数的数量。当可调整参数的数量（有时称为自由度）很大时，容易过拟合
	- 参数的取值。当权重的取值范围较大时，更容易过拟合
	- 训练样本的数量。即使模型很简单，也容易过拟合只包含一两个样本的数据集
6. K折交叉验证：原始训练数据被分成K个不重叠的子集。然后执行K次模型训练和验证来估计训练和验证误差
7. 对于许多任务，深度学习只有在有数千个训练样本时才优于线性模型
# 5. 权重衰减
1. 在训练参数化机器学习模型时，权重衰减（weight decay）是最广泛使用的正则化的技术之一，它通常也被称为L2正则化。这项技术通过函数与零的距离来衡量函数的复杂度
2. L2正则化线性模型构成经典的岭回归（ridge regression）
3. L1正则化线性模型构成经典的套索回归（lasso regression）
4. L2范数的⼀个原因是它对权重向量的大分量施加了巨大的惩罚。这使得我们的学习算法偏向于在大量特征上权重均匀分布的模型。
5. L1惩罚会导致模型将权重集中在一小部分特征上，而将其他权重清除为零。这称为特征选择（feature selection）
6. 非负超参数正则化常数 $\lambda$，较大，对权重约束较大，较小，对权重约束较小
7. 在实例化优化器时直接通过weight_decay指定weight decay超参数。
```python
#偏置参数没有衰减
trainer = torch.optim.SGD([{"params":net[0].weight,'weight_decay':wd},                                 {"params":net[0].bias}],lr=lr)
```
7. 正则化是处理过拟合的常用方法：在训练集的损失函数中加入惩罚项，以降低学习到的模型的复杂度。
# 6. 暂退法（Dropout）
1. 平滑性，即函数不应该对其输入的微小变化敏感。当训练⼀个有多层的深层网络时，在计算后续层之前向网络的每⼀层注入噪声会在输入-输出映射上增强平滑性。
2. 暂退法（dropout）：在前向传播过程中，计算每⼀内部层的同时注入噪声，从表面上看是在训练过程中丢弃（drop out）⼀些神经元，这已经成为训练神经网络的常用技术
3. 关键的挑战就是如何注入这种噪声。⼀种想法是以一种无偏（unbiased）的方式注入噪声。
4. 在标准暂退法正则化中，通过按保留（未丢弃）的节点的分数进行规范化来消除每⼀层的偏差。每个中间活性值h以暂退概率由随机变量h'替换
$$
h' = \begin{cases}0 & p \\ h/(1-p) & 1-p\end{cases}
$$
5. 暂退法仅在训练期间使用。在训练时，Dropout层将根据指定的暂退概率随机丢弃上⼀层的输出（相当于下一层的输入）。在测试时，Dropout层仅传递数据。
```python
net = nn.Sequential(nn.Flatten(), nn.Linear(784, 256), nn.ReLU(),
					# 在第⼀个全连接层之后添加⼀个dropout层
					nn.Dropout(dropout1),
					nn.Linear(256, 256), 
					nn.ReLU(),
					# 在第⼆个全连接层之后添加⼀个dropout层
					nn.Dropout(dropout2), nn.Linear(256, 10))
def init_weights(m):
	if type(m) == nn.Linear:
		nn.init.normal_(m.weight, std=0.01)
net.apply(init_weights)
trainer = torch.optim.SGD(net.parameters(), lr=lr)
```
# 7. 前向传播、反向传播和计算图
1. 前向传播（forward propagation或forward pass）指的是：按顺序（从输入层到输出层）计算和存储中间变量。
2. 反向传播（backward propagation或back propagation）指的是计算神经网络参数梯度的方法。根据微积分中的链式规则，按相反的顺序从输出层到输入层遍历网络，计算和存储神经网络的中间变量和参数的梯度。
3. 反向传播重复利用前向传播中存储的中间值，以避免重复计算，因此需要保留中间值，直到反向传播完成。这些中间值的大小与网络层的数量和批量的大小大致成正比。
# 8. 数值稳定性和模型初始化
1. 梯度爆炸（gradient exploding）：参数更新过大，破坏了模型的稳定收敛
2. 梯度消失（gradient vanishing）：参数更新过小，在每次更新时几乎不会移动，导致模型无法学习
3. 解决（或减轻）上述问题的⼀种方法是进行参数初始化，Xavier（/ˈzeɪviər/）初始化从均值为零，方差为$\sigma^{2} = \dfrac{2}{n_{\mathrm{in}} + n_{\mathrm{out}}}$的高斯分布中采样权重。
4. ReLU激活函数缓解了梯度消失问题，可加速收敛。
