2. 预训练数据集
	- 根据其内容类型进行分类，这些语料库可以划分为：网页、书籍、维基百科、代码以及混合型数据集。
	- 网页
		- 包含丰富多样的文本内容，如新闻报道、博客文章、论坛讨论等
		- `Common Crawl` 
			- 该数据集是一个规模庞大的、非结构化的、多语言的网页数据集，其时间跨度很长，从2008年至今一直在定期更新，总数据量达到PB级别
		- `WanJuan-CC`（万卷 CC）（上海人工智能实验室发布的InternLM2）
			- 该数据集是一个从`Common Crawl`数据中抽取并清洗的高质量英文数据集
	- 书籍
		- `BookCorpus` 该数据集是一个免费小说书籍集合，涵盖了16种不同的主题类型
	- 维基百科
		- 综合性的在线百科全书，提供了高质量的知识信息文章，涵盖历史、科学、文化艺术等多个领域
		- 专业性（结构性和权威性）、多语性（300多种语言）、实时性
		- 通过维基百科的官方提供的下载方式、或Hugging Face
	- 代码
		- 两个主要来源是公共代码仓库（例如`GitHub`）和代码相关的问答平台（例如`StackOverflow`）
		- `The Stack` 
			- 数据来源于`GHArchive`项目中2015年1月1日至2022年3月31日期间的GitHub活跃仓库
	- 混合型数据集
		- `The Pile` 
			- 该数据集是一个大规模、多样化且可公开下载的文本数据集
			- 数据来源非常广泛，包括书籍、网站、代码、科学论文和社交媒体数据等
			- 由22个多样化的高质量子集混合而成，包括`OpenWebText`、维基百科等
3. 微调数据集
	- 指令微调（有监督微调）
		- 自然语言处理任务数据集
			- `P3`：一个面向英文数据的指令微调数据集，由超过270个自然语言处理任务数据集和2,000多种提示整合而成（每个任务可能不止一种提示）
		- 日常对话数据集
			- `ShareGPT`：来源于一个开源的数据收集平台`ShareGPT`，用户可以将自己的各种对话数据通过浏览器插件进行上传
			- `OpenAssistant`：一个人工创建的多语言对话语料库，所有数据都是由用户真实提供
			- `Dolly`：一个英语指令数据集，由`Databricks`员工标注得到
		- 合成数据集
			- 合成数据集通常是使用大语言模型基于预定义的规则或方法进行构建的
			- `Self-Instruct-52K`
				- 人工收集创建了175个种子任务，每个任务包括1个指令和1个实例，每次随机抽取了8个指令作为示例，以此提示GPT-3生成了新的指令，之后在这些已有指令的基础上，继续利用GPT-3生成实例输入及其对应的输出，从而获得了更多数据
			- `Alpaca-52K`
				- 在`Self-Instruct-52K`的175个种子任务上，获得了`52K`个不重复的指令，并根据指令和输入生成了输出，进而构成了完整的实例数据
	- 对齐微调
		- `HH-RLHF`
			- 该数据集包含两大类标注数据，分别关注于大语言模型的有用性和无害性
		- `SHP`
			- 该数据集主要关注模型生成回复内容的有用性，与`HH-RLHF`不同， `SHP`的数据非模型生成，而是人类的回复贴子
		- `CValues`
			- 一个面向中文的大模型对齐数据集，提出了安全性和责任性两个评估标准
4. 代码库资源
	- （1）Hugging Face开源社区
		- Transformers
			- 使用Transformer架构构建模型的开源Python库
			- 易于使用，对所有模型的API进行了统一封装，只需了解三个核心类（模型、配置和分词器），即可快速上手
			- 节省资源，鼓励模型开源共享，减少重复训练
			- 广泛支持，提供数以万计的预训练模型
			- 全周期管理，简化模型训练到部署的过程，支持跨框架模型转换
		- Datasets
			- 该代码库用于高效访问和共享自然语言处理任务相关的数据集，用户仅需一行代码便能加载指定的数据集
		- Accelerate
			- 该代码库是一个旨在简化模型分布式训练和混合精度训练的Python库，专门针对 PyTorch 开发
	- （2）DeepSpeed
		- 微软开发的一个加速深度学习模型训练的高性能库（与PyTorch 兼容），被广泛用于大语言模型的分布式训练
		- 为分布式训练提供了各种优化技术支持，如内存优化（ZeRO技术、 梯度检查点）、数据并行、混合精度训练等
		- 针对模型生成和强化学习分别开发了特制的优化框架：`DeepSpeed-MII`和`DeepSpeed-Chat`
		- `DeepSpeed-MII`
			- 该框架旨在通过提高吞吐量、降低延迟等方式来降低大模型解码生成的运行成本
			- 块状键值缓存：将键值缓存分割成固定大小的块，从而减少了内存碎片化的情况，提升整体的系统吞吐量
			- 连续批处理：在模型的每个前向传播过程中进行独立的调度决策，以实现更细粒度的调度和优化内存效率
		- `DeepSpeed-Chat`
			- 使用方便快捷：该框架简化了类ChatGPT模型的训练和生成过程，可以用简单的脚本实现多个训练步骤
			- 训练通路完整：该框架复现了`InstructGPT`的训练过程，以帮助用户运行完整的训练流程
	- （3）Megatron-LM
		- NVIDIA开发的一款专门为训练大语言模型而设计的深度学习代码库，旨在解决训练过程中的技术挑战，包括显存限制、计算效率以及并行通信问题。
		- 支持多种并行化策略：数据并行，模型并行包括张量并行和流水线并行
		- 支持混合精度训练和Flash Attention功能
