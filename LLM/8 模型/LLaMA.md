2. 核心组件（始终不变）
	- | 组件 | 作用 |
	  |------|------|
	  | Pre-RMSNorm | 层前归一化，稳定训练 |
	  | SwiGLU | 门控激活，提升表达能力 |
	  | RoPE | 旋转位置编码，相对位置感知 |
	  | GQA | 分组查询注意力，省显存 |

	- 关键公式速查
		- **Pre-RMSNorm**：$$\bar{x} = \frac{x}{\sqrt{\frac{1}{d}\sum x_i^2 + \epsilon}} \odot \gamma$$
		- **SwiGLU**：$$\text{SwiGLU}(x) = (\text{Swish}(xW_1) \odot xW_2)W_3$$
		- **RoPE（旋转位置编码）**：$$f(q, m) = q \cdot e^{im\theta} \text{（逐维旋转）}$$
		- **GQA（分组查询）**：
			- $n_{heads}$ 个 Q 头，$n_{kv\_heads}$ 个 K/V 头（$n_{kv\_heads} < n_{heads}$）
			- 每组 Q 共享同一 K/V

3. 关键参数（含义）
	- | 参数 | 说明 |
	  |------|------|
	  | 维度 $d_{model}$ | 隐藏层维度，决定模型宽度 |
	  | 层数 $L$ | Transformer 层数，决定模型深度 |
	  | 注意力头数 | 并行注意力头数量 |
	  | 每头维度 | 每个头的键/值维度（通常 $d_{model}$ / 头数）|
	  | FFN 中间维 | 前馈网络升维后的维度（通常 2.5-4 倍 $d_{model}$）|
	  | 词表大小 | Tokenizer 词汇量 |
	  | 上下文长度 | 最大序列长度 |
	  | 训练数据 | 预训练 token 总量 |
	  | 架构 | Dense vs MoE（稀疏激活）|
	  | RoPE θ | 旋转基频，控制长程依赖 |
	  | 激活参数量 | 推理时实际计算的参数量 |
