1. 时间线
	- | 版本 | 时间 | 核心变化 |
	  |------|------|----------|
	  | Mistral 7B | 2023.10 | 奠基：GQA + RoPE + RMSNorm + SwiGLU |
	  | **Mixtral 8x7B** | **2023.12** | **首个开源 MoE**，8 专家，47B 总参 |
	  | **Mixtral 8x22B** | **2024.04** | **更大 MoE**，8 专家，141B 总参 |

2. 核心组件
	- | 组件 | 配置 |
	  |------|------|
	  | **归一化** | **RMSNorm**（Pre-LN，层前）|
	  | **激活函数** | **SwiGLU** |
	  | **位置编码** | **RoPE** |
	  | **注意力** | **GQA**（Grouped Query Attention）|
	  | **架构** | **Sparse MoE**（稀疏专家混合）|

3. MoE 架构细节
	- | 参数 | Mixtral 8x7B | Mixtral 8x22B |
	  |------|--------------|---------------|
	  | 专家数 | 8 | 8 |
	  | 每 token 激活专家 | **2 个** | **2 个** |
	  | 总参数 | 47B | 141B |
	  | **激活参数** | **13B** | **39B** |
	  | 上下文长度 | 32K | 64K |
	  | 开源协议 | Apache 2.0 | Apache 2.0 |

4. 关键机制
	- **路由机制**：
		- 每层有 8 个 FFN 专家
		- 门控网络（Gating Network）为每个 token 选择 **Top-2 专家**
