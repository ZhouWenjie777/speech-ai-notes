1. 时间线
	- | 版本 | 时间 | 核心变化 |
	  |------|------|----------|
	  | Qwen 1 | 2023.08 | 初代开源，双语能力 |
	  | Qwen 1.5 | 2024.02 | 小模型优化，多尺寸 |
	  | Qwen 2 | 2024.06 | GQA 引入，推理增强 |
	  | Qwen 2.5 | 2024.09 | 18T 数据，长上下文 128K，多模态 |
	  | **Qwen 3** | **2025.04** | **思考/非思考双模式**，MoE，119 语言 |

2. 核心组件（演变）
	- | 组件 | Qwen 1/1.5 | Qwen 2/2.5/3 |
	  |------|------------|--------------|
	  | 归一化位置 | Pre-RMSNorm | Pre-RMSNorm（保持）|
	  | 激活函数 | SwiGLU | SwiGLU |
	  | 位置编码 | RoPE | RoPE（扩展）|
	  | 注意力 | MHA | GQA（Qwen 2+）|
	  | 架构 | Dense | **Dense + MoE**（Qwen 3）|

3. 关键参数（含义）
	- | 参数 | 说明 |
	  |------|------|
	  | 层数 | Transformer 层数（0.6B: 28层，235B: 94层）|
	  | Q/KV 头数 | GQA 分组（如 64/4 表示 64 个 Q 头，4 个 KV 头）|
	  | 专家数 | MoE 中的专家数量（128 个）|
	  | 激活参数量 | MoE 实际计算的参数（如 235B 总参，22B 激活）|
	  | 思考预算 | 推理模式最大思考 token 数 |

4. Qwen 3 关键创新
	- | 特性 | 说明 |
	  |------|------|
	  | **双模式推理** | 思考模式（CoT）+ 非思考模式（快速响应），通过 `enable_thinking` 切换 |
	  | **思考预算** | 可设置最大思考 token 数，控制推理深度 |
	  | **MoE 架构** | 30B-A3B、235B-A22B 等，激活参数 3B/22B |
	  | **119 语言** | 多语言预训练 36T tokens |
	  | **长上下文** | 128K 原生，YaRN 扩展至 1M |
