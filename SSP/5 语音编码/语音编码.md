# 1. 语音编码原理
## 1.1 语音编码分类
1. **编码目的**：减少传输码率或存储量，提高传输或存储效率。
2. **传输码率**：传输每秒钟语音信号所需要的比特数，也称数码率
3. **数字传输系统**：信源-->信源编码-->信道编码-->调制-->传输通道-->解调-->信道解码-->信源解码-->信宿（用户）。
4. **信源编码**：解决有效性，力求最少数码率传递最大信息量。
5. **信道编码**：解决可靠性，力求信号传输过程中不出错。
6. **语音编码**：分为波形编码、参数编码、混合编码。
7. **波形编码**：尽量使重建信号与原始信号在时域波形上保持一致。优点：适应能力强，语音质量好。
8. **参数编码**：一般称作“声码器技术”，通过建模提取代表语音信号特征的参数进行编码，尽量使重建的语音具有足够的可懂度。优点：编码速率低。
9. **混合编码**：上述两种方法的有机结合。
## 1.2 语音压缩原理
1. **基本依据**：语音信号的冗余度和人的听觉感知机理。
2. **时域冗余度**：幅度非均匀分布、样本间的强相关性、浊音语音段的准周期性、语音间隙等等。
3. **频域冗余度**：非均匀的长时功率谱密度（可以压缩或丢弃很多能量很低的频段）、语音特有的短时功率谱密度（由于短时频谱的稳定性，可以只编码相邻帧频谱信息变化的部分）。
4. **人的听觉感知机理**：听觉掩蔽效应、人耳对语音信号相位变化不敏感等等。
# 2. 关键技术
## 2.1 线性预测
1. 基于全极点模型假设，采用时域均方误差最小准则来估计模型参数。
2. 利用LPC技术，可对语音信号进行**样点间的短时相关性**和**相邻基音周期间的长时相关性**的去相关处理，得到预测余量信号，再进行编码，形成激励信号。
## 2.2 合成分析法
1. 使用合成分析法（A-B-S）来求余量信号的编码量化值，可使重建语音误差最小。由于该方法将系统输出引入编码端以调制编码参数，所以也称“闭环法”。
## 2.3 感觉加权滤波器
1. **感觉加权滤波器（PWF）**：根据人耳掩蔽效应设计，把量化噪声“藏”在语音信号强的地方，让人耳听不出压缩带来的失真，从而提高主观听感质量。
# 3. 性能指标和评测方法
## 3.1 主观评价
1. **主观评价**：分为清晰度或可懂度评价和音质评价
2. **可懂度评价（Diagnostic Rhyme Test，DRT）**：衡量通信系统可懂度的ANSI标准之一。
3. **音质评价**：
	- **平均意见得分（Mean Opinion Score，MOS）**：一般采用5级评分标准，通常认为MOS得分在4.0~4.5分为高质量数字化语音。

## 3.2 客观评价
1. **客观评价**：分为时域测度和频域测度
2. **时域测度**：定义为时域波形比较上的失真度，如信噪比（SNR）。
3. **频域测度**：定义为谱失真测度，如对数谱距离测度、LPC倒谱距离测度、巴克谱测度
4. **使用原则**：客观评价用于系统设计阶段，主观评价用于实际听觉效果检验。
# 4. 波形编码
1. **目标**：尽可能还原原始声音的波形，常用于语音和音频压缩。
## 4.1 脉冲编码调制（PCM）
1. **PCM**：最基础的数字音频编码方式，简单说就是：把连续的模拟信号变成离散的数字信号。
	- **采样（Sampling）**：每隔一段时间取一个样本。
	- **量化（Quantization）**：把采样值映射到有限个离散值。
	- **编码（Encoding）**：把量化后的值变成二进制。
2. **均匀PCM**：
	- **原理**：每个采样值用固定的位数表示，比如都用8位或16位。
	- **特点**：简单，但小信号误差大，大信号浪费精度。
3. **非均匀PCM**：
	- **原理**：对信号进行压缩再量化，小信号用更精细的刻度，大信号用粗略的刻度。
	- **方法**：μ律压缩和A律压缩，主要用于电话语音。
	- **特点**：提高小信号的信噪比，整体效果更好。
4. **自适应PCM**：
	- **原理**：根据信号的变化快慢，动态调整量化步长。
	- **特点**：信号变化大时步长大，变化小时步长小，更灵活。
	- **优点**：比前两种更节省比特，适合变化大的信号。
## 4.2 自适应预测编码（APC）
1. **核心思想**：利用信号过去的值来“预测”当前值，只编码预测误差，而不是原始信号本身。
2. **原因**：语音信号相邻采样值之间变化很小，预测误差通常比原始信号小得多。所以可以用更少的比特来编码误差，从而节省带宽。
3. **自适应**：预测器会根据输入信号的变化趋势，动态调整预测系数。
## 4.3 自适应增量调制（ADM）
1. **核心思想**：不传输采样值本身，只传输信号是“变大了”还是“变小了”，而且步长会自动调整。
2. **工作原理**：
	- 每个采样点只传一个比特
		- 1：表示信号比上一个预测值高了
		- 0：表示信号比上一个预测值低了
	- 接收端根据这个比特，逐步“爬”出原始波形。
	- **自适应**：步进幅度Δ随信号平均斜率而变化，斜率大时，Δ增大，反之则减小。
	- 用“爬梯子”的方式还原信号波形，梯子高度还会自动调整。
3. **优点**：编码极简，每采样点只要1个比特
## 4.4 自适应差分脉冲编码调制（ADPCM）
1. **ADPCM**：APC + PCM，既预测误差，又把误差用少量比特（非一位）编码。
## 4.5 子带编码（SBC）
1. **SBC**：也称为频带分割编码，把声音拆成几个“频段”（如低频、中频、高频），每个频段用自己的方式编码。
2. **优点**：符合人耳听觉特性（听觉掩蔽效应）。
## 4.6 自适应变换编码（ATC）
1. **ATC**：利用正交变换去相关作用的特点，通过正交变换，把信号从时域变换到另一个域，使变换域系数集中在一个较小的范围内，降低信号相邻样本间的冗余度。同时使量化字长自适应于每帧语音信号的短时统计特性。
2. **正交变换**：DFT、沃尔什-哈达马变换、DCT、KLT变换。目前，ATC的正交变换都采用DCT。
# 5. 参数编码
## 5.1 线性预测声码器
1. **声码器**：利用参数编码实现语音通信的设备，如通道声码器、共振峰声码器、同态声码器以及线性预测声码器
2. **LPC编码器**：基于全极点声道模型假定，采用线性预测分析合成原理，对模型参数和激励参数进行编码传输。
3. **传输参数**：预测器系数、基音周期、清浊音信息和增益参数。
4. **参数变换**：将预测器系数变换成其他更适合于编码和传输的参数形式。如反射系数、对数面积比，这些参数更稳定、适合量化和编码。
5. **变帧率LPC声码器**：通过度量当前帧参数和上次传送参数间的距离，决定是否传送新的一帧LPC参数。
## 5.2 LPC-10编码器
1. 用 10 个LPC系数建模声道，每帧传一次“喉咙+嘴巴”参数，声音靠对面合成。
# 6. 混合编码
1. **波形编码**：语音音质较高，抗干扰性较好，但比特速率较高且时延较大。
2. **参数编码**：语音音质较低，自然度差，但比特速率大大降低。
3. **合成线性预测编码算法**：多脉冲线性预测编码（MP-LPC）、规则脉冲激励线性预测编码（RPE-LPC）、码激励线性预测编码（CELP）。
## 6.1 CELP
1. **CELP** = “LPC 滤波器 + 双码本激励”，用最小比特传“最像”的噪声序号。
2. **核心理念**：
	- 把声道看成 LPC 滤波器 H(z)
	- 激励不再用单脉冲/单正弦，而是从两个“码本”里挑一段“噪声”
		- **固定码本**（白噪声、稀疏脉冲、代数码本等）→ 提供“粗糙”能量。
		- **自适应码本**（上一帧激励的延迟版本）→ 提供“基音周期”成分。
	- 分析端把成千上万段候选激励送进 H(z)，与原始语音比对，挑“听起来最像”的那一段，只把“序号（codebook index）”和“增益”发出去。
	- 合成端拿到同样的滤波器系数和激励序号，自己再“跑”一遍，就能还原出几乎一样的语音。
# 7. 现代通信编码（EVS）
1. **EVS**：目前涵盖窄带、宽带、超宽带以及全带的最先进的语音频编码器，要求其中的各个单项技术都必须是现有最优秀的技术。
2. **关键技术**：EVS编码器采用基于ACELP的线性预测编码技术、变换编码技术的混合编码方案。
3. **ACELP 线性预测（时域）**
	- 对语音段继续沿用成熟的“**代数码**激励线性预测”。
	- 5.9–24.4 kbps 主要靠它，带内置自适应后置滤波器，音质比 AMR-WB 好 1–1.5 MOS。
4. **MDCT 变换编码（频域）**
	- 32/48 kHz 采样时，对音乐、环境声、清音段切换到 MDCT。
	- 窗长 10/20 ms，TCX（Transform Coded Excitation）结构，频谱噪声整形+全局增益+HVQ 矢量量化。
5. **智能切换与混合**
	- 每 20 ms 帧先做“语音/音乐分类器”，再选 ACELP 或 MDCT
	- 13.2 kbps 以上支持“混合帧”—低频用 ACELP，高频用 MDCT，无缝拼接
6. **AMR-WB**：自适应多速率宽带语音编码（Adaptive Multi-Rate Wideband），用 16 kHz 采样的 ACELP 宽带编码，9 档速率随网切换，让电话从能听清进化到好听，3G/4G 高清语音的幕后标配。
