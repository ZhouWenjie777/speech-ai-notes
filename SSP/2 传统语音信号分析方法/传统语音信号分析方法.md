1. **语音信号具有短时平稳性**：任何语音信号的分析处理须建立在“短时”的基础上，一般将语音信号进行分段，每段称为一“帧”，帧长一般取10~30*ms*。
2. **语音信号分析**：包括时域分析、频域分析、倒频域分析（同态分析）。

# 1. 数字化和预处理
## 1.1 数字化
1. **数字化**：包括预滤波、采样、量化、编码
2. **预滤波**：
	- **混叠**：采样频率小于2倍的信号频率时，高于奈奎斯特频率的信号被重构成低于奈奎斯特频率的信号，导致频谱重叠失真。
	- **目的1**：抑制信号各频域分量中频率超出1/2采样频率的所有分量，防止混叠干扰。
	- **目的2**：避免50*Hz*的电源干扰，因此预滤波是一个带通滤波器。
3. **采样**：
	- 将时间和幅度都连续变化的模拟信号，通过采样和量化，变成时间和幅度都离散的数字信号。
	- **采用频率**：以时间间隔T对连续信号取值，将其变成离散信号，这个过程称为采样，时间间隔T称为采样周期，它的倒数称为采样频率。
	- **采样定理**：当采样频率大于等于2倍的最高截止频率，采样后的离散信号就包含原连续信号的全部信息。
	- 根据采样频率在采样前进行一个低通滤波（预滤波），可防止产生频谱混叠失真。
4. **量化**:
	- 将幅度上连续取值的每个样本转换为离散值表示。
	- **A/D变换**：对信号进行量化，线性A/D变换器目前绝大部分是12位的（即每个采样脉冲转换为12位二进制数字）；非线性A/D变换器则是8位的，与12位线性等效。
5. **编码**：
	- **压缩编码**：对语音数据量进行压缩处理，以减少其传输码率或存储量。
	- **传输码率**：也称数码率或编码速率，表示为传输每秒钟信号所需的比特数，可知越少越好。
	- **编码方法**：波形编码、参数编码、混合编码。
	- **波形编码**：尽量保持波形不变，抗噪性能强，需较高数码率。
	- **参数编码**：又称声码器技术，对语音信号进行分析，提取参数来对参数进行编码，主要从听觉感知的角度注重语音重现。
# 1.2 预处理
1. **预处理**：预加重、加窗、分帧
2. **预加重**：
	- **原因**：平均功率谱受声门激励和口鼻辐射影响，在高频端会跌落，所以求频谱时高频部分比低频部分难求，因此要进行预加重处理。
	- **目的**：提升高频部分，使信号的频谱变得平坦，保持在低频到高频的整个频带中，能用同样的信噪比求频谱，以便频谱分析或声道参数分析。
3. **加窗**：
	- **窗函数**：常用的是矩形窗和汉明窗，窗函数的选择（形状和长度），对于短时分析参数的特性影响很大。
4. **分帧**：可以采用连续分段的方法，一般采用交叠分段的方法，使帧与帧之间平滑过渡

# 2. 端点检测
1. **端点检测**：确定一段语音信号中语音的起始点和结束点的位置。
2. **短时平稳性**：语音在短时间内语音信号的特性基本不变。
3. **短时能量**：一帧语音中语音信号的能量。
4. **短时过零率**：一帧语音中语音信号波形穿过横轴（零电平）的次数。
## 2.1 双门限法
1. **双门限法**：使用二级判决来实现。
	- **一级判决**：以短时平均能量为准。
	- **二级判决**：以短时平均过零率为准。
## 2.2 相关法
1. **短时自相关函数**：引入延迟量，在同一帧内计算相关函数
2. **短时互相关函数**：引入延迟量，在相邻两帧计算相关函数
3. 利用相关函数最大值检测端点，实际上该帧相关函数最大值受该帧能量影响。
## 2.3 能零比双门限法
1. **能零比**：能量和过零率相结合。
	- **能量**：有话帧大于噪声帧。
	- **过零率**：有话帧小于噪声帧。
# 3. 时域分析
1. **时域分析**：分析和提取语音信号的时域参数。
2. **时域参数**：短时能量、短时过零率、短时自相关函数、短时平均幅度差函数，计算这些参数使用的一般是矩形窗或汉明窗。
3. **短时能量**：帧内取样值的平方和
4. **短时平均幅度函数**：帧内取样值的绝对值之和
5. **短时过零率**：一帧语音中语音信号波形穿过横轴（零电平）的次数。
6. **短时自相关函数**：
	- 自相关函数是偶函数
	- 若语音信号是周期的，则自相关函数是同周期的周期函数。
	- 当延迟量k=0时，自相关函数具有最大值。
	- 浊音语音的基因周期可用自相关函数的第一个峰值的位置（k=T）来估算。
7. **短时平均幅度差函数**（AMDF）:
	- 作用类似自相关函数，利用差值，避免乘法，两者之间存在关系表达式。
	- 引入延迟量，帧内取样值差的绝对值之和。
	- 对于周期性的语音信号，AMDF同具周期性，在周期的各个整数倍点上具有谷值。
# 4. 频域分析
1. **频域分析**：即分析语音信号的频域特征，包括频谱、功率谱、倒频谱、频谱包络分析等等。
2. **频域分析方法**：带通滤波器组法、傅里叶变换法、线性预测法。
3. **短时傅里叶变换**：第n帧语音信号的傅里叶变换（离散时域傅里叶变换，DTFT），实际就是窗选语音信号的标准傅里叶变换，在语音信号数字处理中，都是采用离散傅里叶变换（DFT）来替代。
4. **短时功率谱**：短时傅里叶变换的模平方值，也是短时自相关函数的傅里叶变换。
5. **快速傅里叶变换（FFT）**：标准傅里叶变换（DFT）的“算法外挂”——数学结果一模一样，计算量从O(N²)降到O(NlogN)，让“实时”成为可能。
# 5. 线性预测分析
1. **基本思想**：由于语音样点之间存在相关性，所以可以用过去p个样点值来预测现在或未来的样点值。
2. **自回归-滑动平均模型**：简称ARMA模型，同时含有极点和零点。
3. **自回归模型**：简称AR模型，全极点模型。
4. **滑动平均模型**：简称MA模型，全零点模型。
5. 语音信号处理中最常用AR模型。
	- 不考虑鼻音和摩擦音，声道传递函数就是一个全极点模型；另外阶数足够高时，全极点模型可以用来近似极零点模型。
	- 对全极点模型进行参数估计是对线性方程求解，若模型含有零点，则是对非线性方程求解。
6. LPC分析中，线性预测方程组的求解，有自相关法和协相关法两种经典解法。
7. **LPC谱**：求出LPC系数后，得到的语音产生模型的频率响应。
8. **LPC复倒谱**：由LPC系数，递推计算得到的一串呈指数衰减的系数。
