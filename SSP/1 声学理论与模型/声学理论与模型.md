# 1. 声学基础
## 1.1 语音的产生
1. **语音（Speech）**：是声音（Acoustic）和语言（Language）的组合体。
2. **人的说话阶段**：想说阶段、说出阶段（想说阶段大脑中枢的决策，以脉冲形式向发音器官发出指令）、传送阶段（声波）、接收阶段（外耳收集到声波信息，经过中耳放大作用，经过内耳基底膜的振动，激发神经元产生脉冲）、理解阶段（听觉神经中枢）。
3. **语音**：是从肺部呼出的气流通过喉头至嘴唇的器官的各种作用而发出的。
4. **发音器官**：包括肺、气管、喉（包括声带）、咽、鼻、口。喉的部分称为声门，从声门到嘴唇的呼气通道称为声道。
5. **声带**：甲状软骨（喉结）至杓状软骨之间的韧带褶，长度约1*cm*。
6. **基音周期/频率**：声带每开启和闭合一次的时间即声带的振动周期，它的倒数为基音频率。基音频率快则音调高、慢则音调低。
7. **基音**（fundamental tone）：声带开闭一次产生的周期声门波，经声道调制后，变成复杂的复合波，复合波经傅里叶分解后，得到的频率最低、幅度通常最大的正弦分量

## 1.2 语音信号的感知
1. **人耳**
	- **可感知范围**
		- 频率：20 Hz ~ 20 KHz
		- 强度：-5 dB ~ 130 dB SPL
	- **外耳**
		- 耳廓：收集声波、方向定位
		- 外耳道：共振腔，对 2~4 kHz 有天然增益 (~10–20 dB)
	- **中耳**
		- 鼓膜 + 听小骨（锤、砧、镫）：阻抗匹配，将空气声能高效传递至内耳淋巴液
		- 镫骨肌反射：强声时收缩，衰减 30–40 dB，保护内耳、改善言语理解
	- **内耳**
		- 耳蜗：基底膜按频率拓扑分布（**高频近基底，低频近顶**）
		- 毛细胞：内毛细胞（感觉受体）→ 听神经；外毛细胞（主动放大、调谐）
		- 临界带宽：~1/3 oct，决定频率分辨极限
2. **听觉掩蔽效应**
	- 强信号（能量大）附近，弱信号可闻阈提高，甚至完全不可闻
	- **同时掩蔽（频率掩蔽）**
		- 同时存在的强信号（掩蔽音）使邻近频段的弱信号（被掩蔽音）阈值升高
		- 最大掩蔽量：距掩蔽音频率越近、强度越高 → 掩蔽越大；相同声压级下，低频更易掩蔽高频
	- **短时掩蔽（时间掩蔽）**
		- **前掩蔽**：掩蔽音出现前 20–50 ms 内的弱信号被掩蔽
		- **后掩蔽**：掩蔽音结束后 50–200 ms 内的弱信号被掩蔽
		- 机制：神经兴奋/抑制的时间积分效应
	- **应用**
		- 语音/音频编码（MP3、AAC、Opus）利用掩蔽曲线进行比特分配与量化噪声整形
		- 助听器：动态压缩与噪声抑制算法需避开掩蔽区

# 2. 数学模型
1. 语音信号是非平稳随机过程，短段时间内可以认为是平稳随机过程。
2. 语音信号产生的离散时域模型：激励模型、声道模型、辐射模型。
3. **激励模型**：
	- 浊音激励：以基音周期为周期的斜三角脉冲串
	- 清音激励：随机白噪声
4. **声道模型**：包括声管模型和共振峰模型。
5. **共振峰模型**：把声道视为一个谐振腔，共振峰就是这个腔体的谐振频率。
	- 级联型：声道是一组串联的二阶谐振器（对于一般元音，用二阶全极点）
	- 并联型：对于非一般元音及大部分辅音，采用零极点模型。
	- 混合型：
6. **辐射模型**：由辐射引起的能量损耗正比于辐射阻抗的实部，所以辐射模型是一阶类高通滤波器。
7. **语音信号的数学模型**：激励模型、声道模型、辐射模型的串联。

# 3. 特性分析
1. 语音信号属于短时平稳信号，一般认为在10~30*ms*内语音信号特性基本上是不变的，或者变化很缓慢。
2. **语谱图（Spectrogram）**：一种三维频谱，纵轴为频率，横轴为时间，任一频率在给定时刻的强度用相应点的灰度或色调的浓淡来表示。
3. **语谱仪**：记录语谱图的仪器，其本质是滤波器组。
4. **带通滤波器**：分为宽带与窄带。
	- **宽带语谱图**：带宽约为300*Hz*，具有良好的时间分辨率，但频率分辨率较差。
	- **窄带语谱图**：带宽约为45*Hz*，具有良好的频率分辨率，但时间分辨率较差。
5. **声纹**：语谱图上不同的黑白程度，形成的不同纹路。
6. **语音信号波形振幅概率密度函数**：伽马分布、拉普拉斯分布、高斯分布。

# 4. 语音学基本概念
## 4.1 语音学
1. **语音**：人的发音器官发出的声波，具有一定的音色（音质）、音调（频率）、音强（振幅）、音长。
2. **音素（Phoneme）**：语音发音的最小单位，一个音节可以由一个或几个音素构成。
3. 任何语音都有元音和辅音两种音素。
	- **元音**：声带振动，气流顺畅通过不受阻，和鼻腔不发生耦合，声音只从口腔辐射出去。
	- **辅音**：气流被阻不能畅通，声带振动的是浊辅音，声带不振动的是清辅音。
4. **共振峰**：元音的声学特性，元音激励进入声道引起共振，产生一组共振频率（**幅度增益曲线峰值频率**）。实际中，常用前3个共振峰描述语音。
5. 辅音没有明确的共振峰结构，根据阻碍的方法不同分为：塞音、摩擦音、鼻音、颤音等。
6. 基因频率女声高于男声，但区分男女老幼更重要的因素是共振峰频率的高低。

## 4.2 汉语语音学
1. **汉语音系**：简单，音素少（约60个）、音节少（约407个）
2. 汉语由音素构成声母或韵母。
3. **汉语音节**：
	- **声母**：辅音，22个，分为六类：摩擦音、塞音、赛擦音、边音、鼻音、零声母
	- **韵母**：元音，38个，分类三类：8个单韵母，14个复韵母，16个鼻韵母。
	- **声调**：5个，包括阴平、阳平、上声、去声、轻声。
4. 元音、辅音是按音素的发音特征分类的，而声母、韵母是按音节结构分类的。
