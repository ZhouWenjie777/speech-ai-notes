# 1. 同态信号处理
1. **同态信号处理**：也称同态滤波，可实现将卷积关系变换为求和关系的分离处理，即解卷。
2. **倒谱求解方法**：一种是线性预测分析，一种是同态分析处理。
3. **同态信号处理**就是将非线性问题转化为线性问题，按被处理信号分为乘积同态处理和卷积同态处理。
4. **语音信号**可视为声门激励信号和声道响应的卷积。
5. **同态系统**：包括三个子系统。
	- 第一个子系统：将卷积信号转化为加性信号，即求复倒谱。
	- 第二个子系统：对加性信号进行线性变换。
	- 第三个子系统：将加性信号转化为卷积信号，即逆变换。
6. **信号复倒谱**：定义为信号频谱的自然对数的逆傅里叶变换，即进行复对数运算，保留虚部相位信息。
7. **信号倒谱**：定义为信号频谱模的自然对数的逆傅里叶变换，即进行实对数运算，去掉虚部相位信息。
# 2. 小波变换
1. **母小波**：基本小波，在整个实轴上“能量有限”且积分为0。
2. **小波基函数**：母小波经过伸缩、平移后，得到的小波序列。
3. **小波函数选择标准**
	- **支集长度**：指小波函数和尺度函数收敛的速度。算得快（Haar 最短）
	- **对称性**：不飘移（bior 系列）
	- **消失矩**：阶数越高，信号分解后高频分量越少，压缩率越高，能量约集中。多项式信号压缩率高（dbN、symN）
	- **正则性**：越高，重构的信号越光滑。重构曲线光滑（Meyer、Coif）
	- **正交性**：减少数据相关性，降低计算复杂度。无冗余，数据不相关（dbN、symN）
4. **常用的小波基**：Harr小波、dbN小波、symN小波、CoiefN小波、mexh小波、Morlet小波、Meyer小波。
# 3. Teager能量操作
1. **Teager Energy Operator，TEO**：用三个采样点，瞬间算出信号**局部能量**的非线性放大镜——它对幅值大、变化快的成分（冲击、瞬态、高频）特别敏感，而传统能量只是“平方求和。
2. **Teager能量算子**：应用于AM-FM信号，可得出AM和FM分量两者平方的乘积，即幅度包络的平方和瞬时频率的平方的乘积。
3. **AM-FM 信号**：就是调制幅度（音量也在抖）、调制频率（音调也在抖）的波，它的幅度和频率同时随时间慢变化。
4. **能量分离算法（Energy Separation Algorithm，ESA）**：在每一时刻，使用三个相邻采样点做一次TEO，来同时估计中间点的瞬时频率和幅度包络。
# 4. 希尔伯特-黄变换
1. 信号可以分为单分量信号和多分量信号两类。
2. **内禀模式函数（IMF）**：一种单分量信号
	- 过零点和极值点的数目相等或最多相差一个。
	- 由信号局部极大值点相连组成的上包络线与局部极小值点相连组成的下包络线之和为零，即上下包络线对称于零均线。
3. **经验模式分解（EMD）**：处理非线性非平稳信号。
	- 对局部极值点进行三次样条插值拟合，求上、下包络线。
	- 计算上、下包络线的均值曲线。
	- 不断计算、减去均值曲线，提取得到一个IMF分量。
	- 不断提取剩余IMF分量，直到最后剩余分量变成一个单调序列。
4. **系综经验模式分解（EEMD）**：解决IMF模式混叠
	- **模式混叠**：同一个IMF里混叠着两个不同时间尺度的信号，或者同一时间尺度的信号被分解成两个IMF。
	- 1. 将原始信号加上白噪声，获得新的信号
	- 2. 对新的信号进行EMD分解
	- 3. 重复步骤1和2 N次，每次增加不同的白噪声
	- 4. 对分解结果做平均处理，提取得到一个IMF分量。
5. **Hilbert时频谱**：对各个IMF进行Hilbert变换，得到两条幅值和频率序列，叠到同一张坐标纸上，横轴是时间，纵轴是频率，每个格子 (t, f) 的亮度 = 所有IMF在该(t, f)处的幅值贡献之和（或最大值）。
6. **Hilbert边际谱**：Hilbert时频谱对时间的积分，反映每个频率点上幅值的分布，峰值显示“哪段频率能量最多”。
# 5. 经验小波变换
1. **经验小波**：定义在每个傅里叶定义域连续分区上的带通滤波器，包括经验小波母函数和经验小波子函数
2. **经验小波母函数**：针对每一个带通区域，造一个带通滤波器（保证正交/紧支），带通滤波器的性质等价于取出该模式中特定频率的信号，类似IMF。
3. **经验小波子函数**：把母函数平移到各时间段，就得到分析基（跟小波一样伸缩平移，但形状是“数据定制”）。
4. **经验小波变换（EWT）**：先让数据自己告诉在哪些频率段最热闹，再针对这些段量身定做一套小波滤波器，最后做经验小波逆变换得到分量。
5. **改进的EWT**：信号-->傅里叶变换-->**Top-Hat变换**-->频谱划分-->构造经验小波基函数-->经验小波变换
