# 1. 基音周期估计
1. **基音周期估计**：又称基音检测，目标是找出与声带振动频率完全一致或尽可能吻合的基因频率变化轨迹曲线。
2. **基音**：发浊音时声带振动所引起的周期性，其对应的最低频正弦分量。
3. **基音周期**：声带振动频率的倒数。
## 1.1 基音周期估计分类
1. **基于帧的基音周期估计**：通过计算一段语音信号的平均周期来确定基音周期，分为时域算法、频域算法、时频混合算法。
	- **时域算法**：比较直观且运算量小，但抗噪声能力差，容易出现倍频或半频现象。主要算法有自相关法，平均幅度差函数法等。
	- **频域算法**：比较复杂且运算量大。主要方法有倒谱法等。
	- **时频混合算法**：须假定语音在一段内是平稳的，且每段至少包含两个基音周期。
2. **基于事件的基音周期估计**：通过检测声门闭合时刻，测量两个闭合时刻的时间间隔，从而确定基音周期。
	- **基于小波变换**：声门闭合会在语音信号中引起一次阶跃突变，在WT中表现为局部最大。
	- **基于希尔伯特-黄变换**：突变信号的特点是局部时间特征尺度较小，在HHT表现为相邻两极值点的时间间隔很小，且这两极点的幅度差很大。
## 1.2 基于自相关（ACF）
1. **基本思想**：浊音自相关函数在基音周期的整数倍位置上出现峰值，而清音则没有峰值，检测是否有峰值可判断清音或浊音，检测峰值位置可提取基音周期
2. **加窗**：应用矩形窗，窗长至少大于两个基音周期
3. **减少共振峰的“干扰”**：
	- **滤波**：用带宽为60~900*Hz*的带通滤波器，抑制共振峰和50*Hz*电源干扰。
	- **中心削波**：非线性变换，语音信号低幅度部分包含大量的共振峰信息，高幅度部分包含大量的基音信息。
## 1.3 基于幅度差函数（AMDF）
1. **基本思想**：类似自相关函数，不同的是在周期整数倍位置上具有**谷值特性**而不是峰值特性。
2. **加窗**：应用矩形窗，窗长至少大于两个基音周期
3. **减少共振峰的“干扰”**：滤波和中心削波
4. 在基音周期点处AMDF的谷点锐度比ACF的峰点锐度更尖锐，因而估值精度更高，但对语音信号幅度的快速变化比较敏感，因而影响估计的精度。
## 1.4 基于倒谱法（CEP）
1. 倒谱域中基音信息与声道信息可以认为是相对分离的，但并不是完全分离。
2. **算法步骤**：
	- （1）原始语音分帧加窗
	- （2）LPC分析、逆滤波：通过LPC分析获得预测系数，由此构造逆滤波器，再对原始语音进行逆滤波，可获得预测余量信息。
	- 预测余量信息：理论上不包含声道响应信息，却包含完整的激励信息，对预测余量信息进行倒谱分析，可获得更为清晰精确的基音信息。

## 1.5 基于简化的逆滤波跟踪（SIFT）
1. **基本思想**：先对语音信号进行LPC分析和逆滤波，获得预测残差，然后通过自相关滤波器，再做峰值检测，进而获得基音周期。
## 1.6 基于小波变换（WT）
1. **基本思想**：信号小波变换的极值点对应于信号的锐变点或不连续点，对语音信号做小波变换，其极值点对应于声门的开启或闭合点，相邻极值点之距离对应基音周期。
2. **方法比较**：
	- 相比ACF和CEP，准确性最高
	- 计算复杂度：ACF < WT < CEP
	- ACF和CEP是估计信号片段的平均基音周期，小波变换是基于事件的检测法且不建立信号短时平稳性的基础上。
## 1.7 基于倒谱和希尔伯特-黄变换
1. **倒谱包含信息**：声道冲激响应信息和声门激励信息
2. **基本思想**：倒谱法将两个信息由乘性变成加性，然后对倒谱进行经验模式分解，根据分离出来的声门激励信息（即去除IMF的余量信息）获取基音周期。
## 1.8 基于系综经验模式分解（EEMD）
1. **目标**：检测瞬时基音周期，也称为动态基音周期估计。
2. **步骤**：
	- （1）声韵分割：去除无声或清音部分
	- （2）EEMD：生成一系列IMF，结合人工标定基频选择IMF
	- （3）HHT：计算IMF的瞬时频率
	- （4）基频合并：结合人工标定基频。
	- （5）平滑：利用加窗平均，得到最终基频包络
	- （6）取倒：获得基音周期包络
## 1.9 基音周期估计后处理
1. **目的**：去除基音周期轨迹中的偏离点，也称为“野点”。
2. **方法**：中值平滑处理（窗内样点的中值）、线性平滑处理（窗内样点的加权）、组合平滑处理。
# 2. 共振峰估计
1. **元音声源**：具有一个基频和一系列谐波。这些谐波基本都是基频的整数倍，它们的能量随着频率的递增而递减。
2. **共振峰**：基频和谐波通过声腔时，这些谐波被调制，其中被加强的一组谐波群就称为共振峰。
3. **共振峰频率**：浊音短时频谱的包络中有几个明显的凸起点，称为“共振峰”，凸起点处的频率称为共振峰频率，由低到高排列，分别为第一共振峰、第二共振峰、第三共振峰，浊音信号最主要的就是前三个共振峰。
4. **传统的共振峰估计方法**：DFT法、带通滤波器组法、倒谱法、LPC法等等。
5. **LPC法**：传统方法中最有效的一种方法。首先由LPC参数表征声道系统函数，再由该函数求出其频谱，再通过峰值检测求出共振峰频率和带宽。
# 3. 梅尔频率倒谱系数
1. **梅尔频率倒谱系数（MFCC）**:基于人的听觉机理
	- **第一**，人的主观感知频域的划定并不是线性的。
	- **第二**，临界带。按临界带的划分，将语音在频域上划分成一系列的频率群组成了滤波器组，即梅尔滤波器组。
2. **梅尔滤波器组**：每个滤波器具有三角形滤波特性，在梅尔频率范围内，这些滤波器是等带宽的。
3. **MFCC特征参数提取**
	- （1）预处理：包括预加重、加窗、分帧
	- （2）FFT：将时域数据变为频域数据。
	- （3）计算谱线能量：FFT的平方
	- （4）计算通过梅尔滤波器的能量：在频域中相当于把每帧的能量谱与梅尔滤波器的频域响应相乘并相加。
	- （5）计算DCT倒谱：把梅尔滤波器能量取对数后计算DCT。
