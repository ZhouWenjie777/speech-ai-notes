1. 聚合初始化
	- 只要一个 class/struct 同时满足下面几条，就叫“聚合”（aggregate）：
		- 没有用户自己写的构造函数
		- 没有私有/受保护的非静态成员（全部公开）
		- 没有虚函数
		- 没有继承（C++17 起允许继承，但要求所有基类也是聚合，且没有虚函数）
	- “聚合” = Plain Data 包，普通数据包
	- 语法
		- `T obj = {arg1, arg2, ...};   // 以前写法`
		- `T obj{arg1, arg2, ...};      //（C++11起）`
	- struct 和 class 区别
		- 规则一模一样，唯一不同：struct 默认成员都是 public，class 默认 private
		- `struct Point { double x; double y; }; Point p1{3.14, 2.71};`
		- `class Point { public: double x{}; double y{}; }; Point p{1, 2};`
		- 编译器按成员声明顺序按括号里的值依次赋值，少则补 0，多则报错
	- 优点 & 常见用途
		- 不用手写构造函数，就能一次性给所有成员赋初值
		- 数组/标准容器都能直接列表初始化
			- `Point arr[] = { {0,0}, {1,2}, {3,4} };`
			- `std::vector<Point> v = { {0,0}, {1,2} };`
	- 注意：`double x{};` 中 `{}` 称为 “默认成员初始化器”（C++11 新语法），构造函数（或聚合初始化）里没有显式初始化，编译器就自动用 `{}` 把 x 初始化成0.0
