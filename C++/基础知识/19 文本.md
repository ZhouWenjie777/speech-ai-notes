1. 文件操作
	- 文件：
		- 外存数据集合，C++以流形式处理，输入流（从文件到程序）和输出流（从程序到文件）
	- 文件流类：
		- `fstream`派生出`ifstream`（读）、`ofstream`（写）；`fstream`支持读写。
	- 打开文件：
		- `ofstream fout; fout.open("file1");` 或 `ofstream fout("file1");` 指定文件名关联对象
	- 写文件：
		- `fout << x << y;` 类似`cout`，把数据写入关联文件
	- 读文件：
		- `ifstream fin; fin.open("file1");` 或 `ifstream fin("file1");` 
		- 之后`fin >> x`读取
	- 关闭文件：
		- `fout.close();` 断开对象与文件关联，释放资源
	- 控制台：
		- `cin/cout`是`istream/ostream`对象，C++把控制台输入输出视作特殊文件流
2. 文本文件访问流程
	- 包含头文件：
		- `#include <fstream>`，引入文件流操作相关定义。
	- 定义文件流对象：
		- `ofstream outFile;` 或 `ifstream inFile;`，指定操作类型（写或读）。
	- 关联文件：
		- `outFile.open("filename.txt");` 或 `inFile.open("filename.txt");`指定文件名，建立对象与文件的连接。
	- 读写数据：
		- 通过文件流对象的`<<`（写）或`>>`（读）操作，按需处理文件内容
	- 关闭文件：
		- `outFile.close();` 或 `inFile.close();`，释放资源，确保数据正确写入或读取完成
3. 写文件实例
	- 打开：
		- `ofstream outFile; outFile.open("carinfo.txt");` 
		- 失败需检查`outFile.is_open()`
	- 格式化：
		- `outFile << fixed << setprecision(2) << showpoint;`
		- 控制浮点格式，与cout用法一致。
	- 写入：
		- `outFile << 字符串/变量/endl;` 把用户输入的汽车品牌、年份、原价、折扣价写入文本文件
	- 关闭：`outFile.close();` 确保缓冲区刷盘并释放系统描述符
4. 读文件实例
	- 打开：
		- `ifstream inFile; inFile.open(filename);` 
		- 用`is_open()`判失败，`exit(EXIT_FAILURE)`退出。
	- 循环读取：
		- `while(inFile >> value) 或 while(inFile.good()) { ... }` 逐双精度数读入，累加求和。
	- 异常检测：
		- `inFile.eof()`正常到尾；`inFile.fail()`数据类型不符；其他原因用通用fail()诊断。
	- 统计输出：读完后输出条目数、总和、平均值；`inFile.close()`关闭文件。
