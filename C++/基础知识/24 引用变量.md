1. 引用变量
	- 声明：
		- 类型 &别名 = 已有变量; 如 `int &j = k;` j 与 k 代表同一块内存，对 j 的修改即修改 k。
	- 引用传递：
		- 形参写成引用类型，调用时实参必须是变量，函数内对形参的修改直接作用于实参，如 `swapr(a,b)` 完成真交换。
	- 效率：
		- 传引用避免整段数据拷贝，对象/结构体大时比传值高效；
		- 若只读加 const 防误改：void foo(const Big &x)。
	- 引用返回：
		- 在函数返回类型后加 &，表示“返回变量的别名”而非副本；
		- 定义格式：类型 &函数名(形参表) { return 实体; }，例如 `int &index(int i) { return a[i]; }`
		- 由于返回的是别名，调用者可继续对该变量读写，如 `index(2)=25;` 等价于 `a[2]=25;`
		- 关键限制：return 的实体必须具有超出函数作用域的生存期（全局、静态、或由调用者传入的引用），否则返回悬空引用
