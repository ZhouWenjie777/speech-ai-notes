1. 名字空间
	- 用途：
		- 把全局名字分区存放，消除同名冲突；大型项目、第三方库混用时必备
	- 创建：
		- `namespace 名字空间名A { 声明/定义 }`
		- 内部可放变量、函数、类、嵌套名字空间
	- 引用：
		- 完全限定 (`A::gdouble=9.8;`或`B::gdouble=A.gdouble*9.8`）
		- 用 using 声明（`using A::gdouble;`）仅导入单个名字
		- 用 using 编译指令（`using namespace A;`）把空间所有成员一次性导入
	- 作用域差异：
		- using 声明：只影响所在局部作用域，安全且易追踪
		- using 编译指令：文件级则全局可见，易引发二次冲突，头文件中禁止使用
