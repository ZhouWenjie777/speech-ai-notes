1. for循环
	- 格式：
		- for(表达式1; 表达式2; 表达式3) 语句；表达式1初始化，表达式2判定继续，表达式3步进。
	- 字符串逆序：
		- `for(int i = word.size()-1; i >= 0; --i) cout << word[i]`; 下标从尾到头
	- 指针遍历：
		- `for(const char* p = str; *p != '\0'; ++p) cout << (int)*p`; 逐个输出字符ASCII码
	- 范围for（C++11）：
		- for(循环变量 : 序列) 自动按序取值，无需手动下标，可用于数组、列表、string等
	- 递增/递减运算符
		- 前++/--：先改后用，表达式值是修改后对象；后++/--：先用后改，表达式值是原值
	- 复合赋值
		- x += 5等价于x = x + 5；优先级低于算术，结合性右到左
	- 逗号表达式
		- 表达式1, 表达式2, ..., 表达式n依次求值，整体结果为最后一项；常用于for循环头同时更新多个变量
	- 关系运算
		- 整型、浮点、字符、bool可直接用< > \==；优先级低于算术，高于赋值，左结合，结果bool
	- 字符串比较
		- C++ string：支持== < >等运算符，按字典序逐字符比较
		- C字符数组：用`cstring`库`strcmp(s1,s2)`；返回`=0 s1=s2，<0 s1<s2，>0 s1>s2`；不可直接用\==比较地址
2. while循环
	- 格式：
		- while(测试条件) { 循环体; }；先判后执行，条件真继续，适用于循环次数未知场景
	- 遍历C串：
		- `while(name[i] != '\0') { cout << name[i] << ':' << int(name[i]) << endl; ++i; }` 无需事先知道长度
	- 回显+计数：
		- `cin.get(ch)`能读空格回车，`while(cin.get(ch) && ch != '#') { cout << ch; ++count; }` 用自定义结束符避免丢失空白。
	- EOF结束：
		- Unix Ctrl+D / DOS Ctrl+Z 触发`cin.get(ch)`返回false，可写`while(cin.get(ch)) { ... }` 实现“读到文件尾”。
3. do-while循环
	- 格式：
		- do { 循环体; } while(测试条件); 先执行后判断，至少跑一轮，适合“出口条件”场景
	- 示例：
		- 不断`cin.get(ch)`回显并++count，直到读入'#'结束；#仍被输出，体现“至少执行一次”特点
	- 对比while：
		- while可能0次执行，do-while保证1次；当循环体需先获得条件变量值时常用
