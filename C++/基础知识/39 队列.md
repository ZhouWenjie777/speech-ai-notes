1. Queue 类型接口（基于链表实现）
	- 队列语义：FIFO 线性表，插入（enqueue）在表尾，删除（dequeue）在表头。
	- 类型参数：模板或 `typedef` 方式定义 `Item`，用户可在使用前指定实际数据类型
	- 存储设计：
		- 单链表，节点结构：`struct Node { Item item; Node *next; };`
		- 额外维护两个指针 `front`（头节点，出队端）与 `rear`（尾节点，入队端），以及计数器 `items`与容量上限`qsize`
	- 构造函数：`Queue(int qs = QSIZE)`用默认/给定大小初始化空队列，分配头节点或空指针，置 `items = 0`
	- 容量查询：
		- `bool isempty() const  ——  items == 0`
		- `bool isfull() const  ——  items == qsize`
		- `int queuecount() const  —— 返回当前  items`
	- 核心操作：
		- 入队  `enqueue(const Item &item)`
			- 若已满返回 `false`；否则新建节点挂到 `rear->next`，更新 `rear = newNode`，`++items`，返回`true`
		- 出队 `dequeue(Item &item)`
			- 若为空返回 `false`；否则把 `front->item`拷给引用参数，移动 `front` 到 `front->next`，删除旧头节点，`--items`，返回`true`
	- 析构函数：循环释放所有节点，防止内存泄漏
	- 复杂度：所有操作均为 `O(1)`，无需整块搬移数据；容量上限 `qsize` 可在构造时指定，达到上限后拒绝入队
	- 清空辅助：可在类内额外提供`clear()` 方法，重复调用 `dequeue`直至`isempty()`为真，方便重用队列对象
2. 队列（类方法的实现）
	- 构造函数初始化列表
		- `Queue::Queue(int qs) : qsize(qs), front(nullptr), rear(nullptr), items(0) { }`
		- 常量数据成员 `qsize` 必须在初始化列表里完成“初始化”，函数体内部再写 `qsize = qs;` 属于赋值，对常量非法
		- 指针型成员建议列表初始化为 `nullptr`，避免未定义野指针
	- 入队 `enqueue(const Item &item)`
		- `if (isfull()) return false;`
		- `Node *add = new Node{ item, nullptr };   // 新建节点`
		- `if (front == nullptr)                    // 空队列`
			- `front = add;                        // 新节点既是头也是尾`
		- `else`
			- `rear->next = add;                  // 原尾节点指向新节点`
		- `rear = add;                             // 更新尾指针`
		- `++items; return true;`
	- 出队 `dequeue(Item &item)`
		- `if (isempty()) return false;`
		- `item = front->item;        // 传出数据`
		- `Node *temp = front;        // 保存旧头`
		- `front = front->next;       // 头指针后移`
		- `delete temp;               // 释放旧头`
		- `--items;`
		- `if (items == 0) rear = nullptr; // 队列变空，尾指针也要清零`
		- `return true;
	- 析构函数
		- `Queue::~Queue() {while (front != nullptr) {Node *tmp = front;front = front->next;delete tmp;}}`
		- 循环释放所有节点，防止内存泄漏；空队列时 `front == nullptr`，循环体不会执行
	- 状态查询（内联建议）
		- `bool Queue::isempty()  const { return items == 0; }`
		- `bool Queue::isfull()   const { return items == qsize; }`
		- `int  Queue::queuecount() const { return items; }`
