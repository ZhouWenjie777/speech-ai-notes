1. C++11 核心回顾
	- （1）统一初始化（initializer list）
		- 语法：`T obj { arg1, arg2, ... };`  兼容数组、容器、类类型
		- 示例
			- `int x{3};                          // 内置类型`
			- `double a[]{1.1, 1.2, 1.3, 1.4, 1.5};`
			- `int* p = new int[3]{1, 2, 3};      // 堆数组亦可列表初始化`
			- `vector<int> v{4, 6, 1};            // 容器`
			- `A a{1, 2};                         // 类：匹配构造函数`
		- 优势
			- 禁止窄化转换（`int x{3.14}` 编译错误）
			- 消除“最令人头疼的解析”歧义
	- （2）`auto` 类型推导
		- 让编译器用初始化表达式推断变量类型
		- `auto abc = 3.14;   // double`
		- `auto it = v.begin(); // vector<int>::iterator`
		- 规则
			- 必须立即初始化
			- 会去掉顶层 `const`；需要保留写 `const auto` / `auto&`
	- （3）`decltype` 类型推断
		- 根据任意表达式得到类型，不计算值
			- `int a = 1, b = 2;`
			- `decltype(a + b) c;   // c 为 int（a+b 类型）`
		- 常用于尾置返回与模板元编程
			- `template<typename T, typename U>`
			- `auto f(T t, U u) -> decltype(t + u) { return t + u; }`
	- （4）新整型 `long long`
		- 至少 64 位；对应字面量加 `LL` / `ll` 后缀：`123LL`
	- （5）类内成员初始化
		- 在声明处给默认值；构造函数未显式初始化则使用该值
		- ```cpp
			  class Notes {
				  int k = 0;                    // 类内初值
				  double x{0.01};               // 统一初始化亦可
				  std::string st{"oh"};         // 非静态成员
			  public:
				  Notes() = default;            // k/x/st 用上方值
				  Notes(int kk) : k(kk) {}      // 仅 k 被覆盖
			  };
			  ```
		- 规则
			- 仅适用于非静态数据成员
			- 与初始化列表不冲突，列表优先
	- （6）委托构造（constructor delegation）
		- 一个构造函数在初始化列表里调用**同类**的另一个构造函数，避免代码重复
		- `Notes::Notes() : Notes(0, 0.01, "oh") { /* do other stuff */ }`
		- `Notes::Notes(int k) : Notes(k, 0.01, "oh") {/* do other stuff */}`
		- 注意
			- 委托语句必须是初始化列表唯一成员（不能再初始化其他成员）
			- 执行顺序：先被委托者，再执行委托者函数体
	- （7）显式类型转换运算符
		- 在转换函数前加 `explicit`，禁止隐式对象转换
		- ```cpp
			  class Rational {
				  int num, den;
			  public:
				  explicit operator double() const {return double(num)/den;}
			  };
			  Rational r(3,4);
			  double x = r;          // ❌ 隐式转换被禁止
			  double x = double(r);  // ✅ 显式允许
		  ```
	- （8）新增默认函数（=default / =delete）
		- `Notes() = default;` 让编译器生成默认实现，同时保留“平凡”属性
		- `Notes(const Notes&) = delete;` 显式禁用拷贝
	- （9）移动语义（提及）
		- 编译器自动生成移动构造函数与移动赋值运算符，前提：无自定义析构/拷贝/赋值，且所有成员可移动
		- `Notes(Notes&&) noexcept = default;`
		- `Notes& operator=(Notes&&) noexcept = default;`
	- （10）总结
		- C++11 让初始化、类型推导、类构造更统一、更安全，为后续移动、lambda、并发等特性打下基础；
		- 牢记“列表初始化防窄化、 `auto` 必须初始化、 `decltype` 只拿类型
2.  C++11 其余核心特性
	- （1）右值引用与移动语义
		- 语法：`T&&` 只能绑定到临时对象（`xvalue`）
		- 用途
			- 移动构造/赋值接管资源，避免深拷贝
			- 完美转发：`template<class F, class... Args> auto wrapper(F&& f, Args&&... args) -> decltype(f(args...)) { return f(std::forward<Args>(args)...); }`
	- （2）`std::move`
		- 将左值强制转成右值引用，**仅类型转换**，不移动数据；
		- 典型用法：`return std::string(std::move(localStr));`
	- （3）可变参数模板（Variadic Template）
		- `template<typename... Ts>`
		- `void print(Ts... args) { (std::cout << ... << args) << '\n'; // C++17 折叠表达式 }`
	- （4）初始化列表 `std::initializer_list<T>`
		- 让自定义容器也支持`{ }`语法：`Vec(std::initializer_list<T> l) : Vec(l.begin(), l.end()) {}`
	- （5）`static_assert`
		- 编译期断言，失败则报错：`static_assert(sizeof(void*) == 8, "64-bit only");`
	- （6）`nullptr` 与 `std::nullptr_t`
		- 专门表示空指针，避免与整数 `0` 混淆
		- 重载分辨率：`f(int)` vs `f(void*)` 不再歧义
	- （7）基于范围的 `for（Range-based for）`
		- `for (auto& x : container) { ... } `
		- 等价调用 `begin(container)` / `end(container)`
	- （8）`Lambda` 表达式
		- `[捕获](参数) mutable -> 返回类型 { 函数体 }`
		- 编译器生成**匿名函数对象**，可捕获局部变量、可赋值、可泛型：
		- `auto cmp = [](const auto& a, const auto& b){ return a < b; };`
	- （9）函数对象包装器
		- `std::function<R(Args...)>`  通用多态函数封装
		- `std::bind`  偏绑定 + 占位符 `_1, _2...`
		- `std::ref` / `std::cref` 引用包装，解决值拷贝问题
	- （10）类型别名模板
		- `template<typename T>`
		- `using Vec = std::vector<T>;        // 带模板参数的 typedef`
		- `Vec<int> v;                        // = vector<int>`
	- （11）枚举类（强类型枚举）
		- `enum class Color : unsigned char { Red=1, Green, Blue };`
		- 不会隐式转整型，可指定底层类型，可前向声明
	- （12）默认/删除函数控制
		- `Foo() = default;                   // 强制编译器生成`
		- `Foo(const Foo&) = delete;          // 禁止拷贝`
		- 也可用于特殊成员：移动构造、赋值、析构
	- （13）常量表达式 `constexpr`
		- `constexpr int sq(int x){ return x * x; }`
		- 允许在**编译期**求值，可用于数组大小、模板实参
		- C++11 要求函数体**单一 return**（C++14 起放宽）
	- （14）用户定义字面量（原始形态）
		- 字符串：
			- `const char* s = R"(C:\path\file.txt)";  // 原始字符串，无转义`
		- 自定义（需实现后缀）：
			- `std::string operator "" _s(const char* p, std::size_t n){ return std::string(p,n); }`
			- `auto str = "hello"_s;                   // 返回 std::string`
	- （15）线程支持库 `<thread>`（头回出现）
		- `std::thread t([]{ std::cout << "并发\n"; });`
		- `t.join(); // 等待完成`
		- 还有 `mutex、lock_guard、condition_variable、async、future` 等
	- （16）时间工具 `<chrono>`
		- `auto t0 = std::chrono::steady_clock::now();`
		- `auto dur = std::chrono::duration<double, std::milli>(t1 - t0).count();`
	- （17）无序容器（哈希表）
		- `std::unordered_map / unordered_set / unordered_multimap / unordered_multiset`
		- 平均 O(1) 查找，Key 需支持 hash 函数
	- （18）正则库 `<regex>`
		- `std::regex  re(R"(\d{3,4})");`
		- `bool ok = std::regex_match("1234", re);`
	- （19）[[44 智能指针 | 智能指针]]
		- `unique_ptr、shared_ptr、weak_ptr、make_shared、make_unique (C++14 补回)`
	- （20）编译器提示
		- 使用以上特性需 **-std=c++11**（GCC/Clang）或 **/std:c++11**（MSVC）
		- 个别特性（如`<thread>`、`<regex>`）需链接 `pthread` 或系统库
