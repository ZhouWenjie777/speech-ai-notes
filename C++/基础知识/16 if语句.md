1. if语句
	- 格式：
		- if(表达式) 语句; 表达式为true执行语句，否则跳过；无else时称为“单分支”
	- if-else：
		- if(表达式) 语句1; else 语句2; 二选一，else与之前最近且无else的if配对
	- 嵌套：
		- then/else子句仍是if语句形成多级判断；
		- else悬挂问题靠“最近未配对”原则自动绑定
	- 歧义示例：通过缩进或加花括号显式划分层次，增强可读性并避免逻辑错误
2. 逻辑表达式
	- 运算符：
		- !（非）> &&（与）> ||（或），均低于关系运算；先左后右短路求值，一旦整体结果确定立即终止。
	- 短路示例：
		- a||++b，若a非0则++b不执行；++b未发生，保持原值。
	- 运算对象：
		- 任意标量，0视为false，非0视为true；结果类型bool，true/false可隐式转1/0
	- 实例：
		- is_int(double x)用x<=INT_MAX && x>=INT_MIN判断double值是否落在int合法范围，避免强制转换溢出
3. 条件表达式
	- 格式：
		- 表达式1 ? 表达式2 : 表达式3；三元运算符，紧凑替代简单if-else。
	- 执行：
		- 表达式1为true，结果是表达式2；否则是表达式3；类型自动推导为二者中更通用的那个
	- 应用：
		- 赋值：`max = a > b ? a : b;` 一行搞定变量赋值
		- 输出：`cout << (a > b ? "true" : "false");` 根据条件动态输出内容。
	- 注意：表达式2和3类型需兼容，否则编译报错；嵌套使用时需加括号明确优先级
