1. 参数的默认值
	- 声明处指定：
		- 返回类型 函数名(形参表 = 默认值); 
		- 例 `void print(int value, int base = 10);` 
		- 调用 print(20) 等价 print(20,10)。
	- 使用规则：
		- 默认参数必须**从右向左**连续出现；非默认参数不能在默认参数右侧，
		- 如 `int SaveName(char* first, char* second = "", char* third = "", char* fourth = "");` 合法。
	- 多文件差异：不同源文件可在各自原型中给出不同默认值，实现同一函数针对不同模块的“定制”，编译器按调用处可见原型决定实际缺省值。
	- 示例：`char* left(const char* str, int n = 1);` 省略第二参时自动取1，实现“缺省取首字符”功能；动态返回缓冲区，调用者必须 `delete []` 释放。
