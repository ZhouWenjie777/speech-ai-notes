1. 结构
	- 声明：struct 结构名 { 成员列表; }; 仅描述布局，不分配内存
	- 定义变量：结构名 变量名; 如 `book book1`; 数组：`book bookarray[10]`
	- 初始化：`book book1 = {"C Primer Plus", "Stephen Prata", 89.0}`; 数组同理按行赋初值
	- 访问：变量名.成员名; 如 `book1.author`; 数组元素 `bookarray[2].title`
	- 赋值：同类型结构可整体赋值，`bookarray[0] = bookarray[1]`; 各成员依次复制
	- 位字段：成员后:位数; 如 `unsigned int SN:4`; 节省空间，常用于映射硬件寄存器
