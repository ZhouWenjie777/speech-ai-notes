1. 文本正则化（同Glow-TTS，`english_cleaners` 差异点）
	- 不走 ARPAbet 路线，直接调用第三方 phonemizer 库
	- `phonemized = phonemize(text, backend='espeak', language='en-us', strip=True, preserve_punctuation=True, with_stress=False)`
	- 返回 IPA 音素序列，如 `"hello world" → "həˈloʊ wɜrld"`
	- 后续流程
		- 按空格分词 → 按 IPA 表查符号 → id
		- symbols 总长 178：
			- 1 \_pad 
			- 16 标点（`.,!?`等）
			- 52 字母（`a-zA-Z`
			- 109 IPA 音素（含 `ʃ, ɜ` 等）
		- 得到 `int` 序列后直接喂入 `TextEncoder` 的 `embedding` 层
	- 与 Glow-TTS 对比
		- Glow-TTS 用 84 个 ARPAbet
		- VITS 用 109 个 IPA，集外词无需回退到 grapheme，端到端压力更小，但依赖系统级 `espeak-ng` 后端
2. 摘要核心（并行端到端 TTS）
	- Variational Inference with adversarial learning for end-to-end Text-to-Speech：基于变分推断与对抗学习的端到端文本到语音合成系统
	- 问题现状
		- 近期单阶段并行 TTS（FastSpeech、Glow-TTS 等）训练简化、推理快，但音质仍落后于两阶段系统（Tacotron2 + WaveNet）  
	- VITS 方案
		- 变分推断 + 归一化流（Variational Inference + Normalizing Flows）
			- 增强生成表达能力，精确建模似然
		- 对抗训练（Adversarial Training）
			- 波形判别器引导生成器，提升自然度
		- 随机时长预测器（Stochastic Duration Predictor）
			- 显式对“文本→多种节奏/音高”一对多关系建模，采样不同时长 → 语音更具多样性
	- 结果（LJSpeech 单说话人）
		- MOS 与**真值录音无统计差异**，**超越当时最好公开 TTS**（含两阶段）
		- 单模型、单阶段训练、完全并行采样，首次实现“**快+高质+多样性**”三兼顾
3. 引言关键信息
	- 两阶段痛点
		- 仍需顺序训练/微调：第二阶段必须用第一阶段生成样本，误差累积
		- 手工中间特征（mel/linguistic）限制模型利用更优的隐表示进一步提升质量
	- 单阶段并行进展
		- FastSpeech 2s、EATS 已验证“短片段 + 辅助 mel 解码器”可实现端到端训练，但音质仍落后两阶段
	- VITS 突破
		- **单网络、单阶段、完全并行**，首次在 MOS 上**超越当前最佳两阶段系统**（Glow-TTS+HiFi-GAN）
		- 技术组合：
			- 条件 VAE + Normalizing Flows：提升先验表达能力，精确似然
			- 对抗训练：波形判别器驱动，细节自然度↑
			- 随机时长预测器：显式建模一对多（节奏/音高变异），采样多样性↑
	- 结果
		- LJSpeech主观评测MOS与**真值无统计差异**，采样速度↑，代码与demo已开源
	- 实现技巧
		- 短音频片段训练：显存受限下用 1-2 s 切片，辅以 mel-decoder 辅助任务，稳定文本表征学习
		- 先验流 + 对抗：Flow 负责精确似然，GAN 负责感知细节，两者互补
		- 随机时长：对每音素采样时长，而非硬对齐，天然支持同句多节奏
	- 定位 
		- 首次把**VAE+Flow+GAN+随机时长**打包进单一并行网络，端到端训练，推理一步出波形
		- 代码与演示页已公开，后续被广泛作为并行 TTS 基准
4. 条件 VAE 框架（训练视角）
	- VAE 全称：**Variational Autoencoder（变分自编码器）**
	- 目标函数
		- 最大化 the evidence lower bound，ELBO$$
			\log p_\theta(a|c) \geq \mathbb{E}_{q_\phi(z|a)}\left[\log p_\theta(a|z)\right] - \text{KL}\left[q_\phi(z|a) \parallel p_\theta(z|c)\right]
			$$
		- 负 ELBO 即总损失：$$
			\mathcal{L} = \underbrace{-\log p_\theta(a|z)}_{\text{重建}} + \underbrace{\text{KL}\left[q_\phi(z|a) \parallel p_\theta(z|c)\right]}_{\text{KL 正则}}
			$$
	- 重建损失
		- 目标：mel-spectrogram（非波形）
		- 过程：latent z → 解码器 → 时域波形 → **STFT + mel 投影** → 预测 mel
		- 形式：L1 距离，等价于 Laplace 似然（忽略常数）
		- 感知友好：mel 尺度逼近人耳响应；STFT/mel 无参，仅训练期使用
	- KL 散度
		- 后验：$q_\phi(z|a)=\mathcal{N}(z;\mu(a),\sigma(a))$，输入为**线性谱**以提供高分辨率信息
		- 先验：$p_\theta(z|c)=\mathcal{N}(f_\theta(z);\mu(c),\sigma(c))$，$c=[\text{phoneme},\text{alignment }A]$
		- 先验用 **normalizing flow** $f_\theta$ 提升表达力，使简单高斯可逆变换为复杂分布
	- 窗口生成器训练
		- 不整句 upsample，只在短窗内重建，节省显存，支持端到端大批量训练
		- 类似 FastSpeech 2s / EATS 短片段策略，梯度噪声兼正则效果
5. 对齐与随机时长预测器
	- MAS 在 VITS 中的适配
		- 目标：最大化 ELBO 而非精确似然 → 式 (6) 等价于最大化 latent 变量 z 的 log-likelihood
		- 因此可直接复用原始动态规划 MAS，无需修改代码
		- 输出：硬单调对齐 A，行求和即得每音素时长 $dᵢ$
	- 随机时长预测器动机
		- 人类同一句语速/节奏可变；确定性预测器只能给唯一值
		- 解决方案：**flow-based 生成模型**，从噪声采样即可得不同时长序列
	- 技术实现（式 7）
		- 变分解量化 & 数据增广：引入辅助变量 $u∈[0,1)$、$v$（与 $d$ 同维）
			- $d−u$ 为正实数，支持连续流变换
			- $[v;d]$ 通道拼接，提升 latent 维度，避免标量可逆性瓶颈
		- 近似后验 $q(u,v|d,c)$ 可 tractable，负 ELBO 即 $L_{dur}$
		- stop-gradient 切断梯度，防止时长训练干扰其余模块
	- 采样推理
		- 噪声 $ε$ → 逆变换 → 连续时长 → 取整 → 整数帧长
		- 同一文本多次采样可得不同节奏，实现“一对多”自然变异
6. VITS 对抗训练与总损失
	- GAN全称：**Generative Adversarial Network（生成对抗网络）**
	- 判别器 D
		- 目标：区分真实波形 y 与生成波形 G(z)
		- 损失：Least-Squares GAN
			- D 损失：$\mathcal{L}_{\text{adv}}(D)=\mathbb{E}[(D(y)-1)^2]+\mathbb{E}[D(G(z))^2]$
			- G 损失：$\mathcal{L}_{\text{adv}}(G)=\mathbb{E}[(D(G(z))-1)^2]$
	- 特征匹配损失 $\mathcal{L}_{\text{fm}}(G)$
		- 定义：生成样本与真实样本在 D 的各隐藏层特征图 L1 距离$$
			  \mathcal{L}_{\text{fm}}(G)=\mathbb{E}\sum_{l=1}^{T}\frac{1}{N_l}\|D^{(l)}(y)-D^{(l)}(G(z))\|_1
			$$
		- 作用：替代像素级 L1，让 G 匹配真实分布的**中间统计量**，提升细粒度自然度
	- 总损失（式 11）
		- $$
			\mathcal{L}_{\text{vae}} = \underbrace{\mathcal{L}_{\text{recon}}}_{\text{mel-L1}} + \underbrace{\mathcal{L}_{\text{KL}}}_{\text{先验正则}} + \underbrace{\mathcal{L}_{\text{dur}}}_{\text{时长 ELBO}} + \underbrace{\mathcal{L}_{\text{adv}}(G)}_{\text{GAN}} + \underbrace{\mathcal{L}_{\text{fm}}(G)}_{\text{特征匹配}}
			$$
	- 训练策略
		- VAE 与 GAN 联合优化；$\mathcal{L}_{\text{fm}}$ 仅更新 G，不更新 D，避免判别器退化
		- 各分量权重 1:1:1:1:1，无需额外调参即可稳定收敛
7. VITS 模型组件一览
	- Posterior Encoder（训练专用）
		- 非因果 WaveNet 残差块堆叠 → 线性投影输出 μ, σ
		- 输入：线性谱（高分辨率）
		- 多说话人：全局 speaker embedding 注入残差块
	- Prior Encoder（推理也用）
		- Transformer Encoder（相对位置编码）→ 线性层输出 μ, σ
		- Normalizing Flow：仿射耦合层堆叠，**体积保持**（Jacobian=1），简化计算
		- 多说话人：speaker embedding 加在残差块输入
	- Decoder（HiFi-GAN V1 生成器）
		- 转置卷积上采样 + MRF（Multi-Receptive Field Fusion）
		- 输出：24 kHz 原始波形
	- Discriminator（训练专用）
		- Multi-Period Discriminator（HiFi-GAN）
		- 多组不同周期窗口的 Markov 子判别器，捕获不同频率模式
	- Stochastic Duration Predictor
		- 空洞 & 深度可分离卷积残差块
		- Neural Spline Flows（单调理性二次样条）→ 连续可逆变换，提升表达能力
		- 输入：音素隐藏 $h_{text}$；输出：连续时长 → 取整得帧级长度
	- 整体流（单说话人）
		- 文本 → Prior Encoder → 先验 μ,σ + Flow → z
		- 线性谱 → Posterior Encoder → 后验 μ,σ → z
		- z → Decoder → 波形
		- D 判别真假，G 用对抗+特征匹配损失优化
		- 对齐 A 由 MAS 在 latent 空间自监督得到，用于时长标签
8. 预处理与训练配置
	- 预处理
		- 后验编码器输入：线性谱（STFT: 1024/1024/256）
		- 重建目标：80-band mel（线性谱 → mel 滤波库）
		- 文本输入：IPA 音素序列，开源工具转换，空白 token 分隔（同 Glow-TTS）
	- 训练策略
		- 优化器：AdamW (β1=0.8, β2=0.99, wd=0.01)
		- LR：初始 2×10⁻⁴，每 epoch ×0.999^(1/8) 衰减
		- Windowed Generator：仅随机 32 帧 latent 片段 + 对应音频送入解码器，显存↓，训练加速
		- 混合精度：4×V100，单卡 batch=64，总步数 800k
	- 硬件与时间
		- 4 V100 × 800k step ≈ 7 天（LJSpeech 24 h 数据），与 Glow-TTS 同级
		- 窗口训练使单卡可扛 64 batch，无额外梯度累积
