1. mel vs MFCC
	- | 维度 | mel 频谱 | MFCC |
	  |---|---|---|
	  | **本质** | 人耳 mel 滤波器组对数能量 | 对 mel 能量再做 DCT 得到的倒谱系数 |
	  | **维度** | 常见 40/80 维（滤波器个数） | 通常 13 维（+Δ+ΔΔ→39） |
	  | **相关性** | 高，帧间冗余大 | 低，近似去相关 |
	  | **信息量** | 高，保留更多音色细节 | 低，仅保留包络 |
	  | **计算** | FFT → mel 滤波 → log | 同mel + DCT |
	  | **ASR 用法** | 神经网络端到端首选 | 传统 GMM-HMM 首选 |
	  | **优点** | 细节丰富，易重构 | 维度小，抗信道干扰 |
	  | **缺点** | 维高、冗余 | 丢失细节，不适合重构 |
